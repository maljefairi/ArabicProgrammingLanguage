# كود البرمجة باللغة العربية الفصحى

التجاوز في البرمجة باللغة العربية يمكن تنفيذه من خلال استخدام عبارة "يجاوز". هذه العبارة تلغي تنفيذ الدالة الأصلية بالكامل وتقوم بإعادة كتابة الدالة بشكل جديد. لنرى كيف يمكن تطبيق ذلك:

## مثال على استخدام التجاوز في البرمجة باللغة العربية

لنفترض أن لدينا الكائن الأب "حيوان" وهو يحوي الوظيفة "صوت". بإمكان الكائن الابن، مثل "القط"، التجاوز لهذه الوظيفة لإعطائها تصرف فريد.

في الكود التالي، نرى تطبيق لهذا المفهوم:

```arabic
صنف حيوان {
   دالة صوت() {
      اكتب("الحيوان يصدر صوت")
   }
}

صنف قط يرث من حيوان {
   يجاوز دالة صوت() {
      اكتب("القط ينقر")
   }
}

كائن توم من قط
توم.صوت() // الناتج سيكون: القط ينقر
```

في هذا المثال، القط هو الصنف الابن الذي يتجاوز دالة "صوت()". عند استدعاء "توم.صوت()", الدالة المتجاوزة في الصنف القط تتجاوز الدالة الموجودة في الصنف حيوان، وبالتالي يتم طباعة "القط ينقر" بدلاً من "الحيوان يصدر صوت".
