# التجاوز في البرمجة باللغة العربية

التجاوز (Override) هو أحد أساسيات البرمجة الموجهة للكائنات وهو الصفة التي تتيح لك تغيير سلوك أو خاصية موروثة من الصف الأب.

فلنفترض بأن لدينا صف يدعى "حيوان" ولدينا صف آخر يدعى "قطة" يرث من الصف "حيوان".

في اللغة البرمجة بالعربية، يمكننا تطبيق ذلك كما يلي:

```
صف حيوان {
  دالة صوت() {
    أطبع("الحيوان يصدر صوت...")
  }
}

صف قطة يرث حيوان {
  دالة صوت() تجاوز {
    أطبع("القطة تقول مياو")
  }
}
```

في الكود السابق، الصف "قطة" يجاوز الدالة "صوت" في الصف "حيوان". عندما نستخدم هذه الطريقة، البرنامج الخاص بنا سوف يتجاهل الدالة "صوت" في الصف الأب "حيوان" وسوف يستخدم الدالة المجاوزة في الصف الابن "قطة".
