أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

أعرض لكم هنا مثال على كيفية إعداد واستخدام الطبقات والواجهات في البرمجة باللغة العربية الفصحى. في هذا المثال، سوف نقوم بإنشاء طبقة تدعى "المدينة" التي تحوي خاصيتين، اسم المدينة و عدد السكان، و واجهة تدعى "المعلومات" تحوي وظيفة واحدة لطباعة المعلومات:

الكود التالي يقوم بإنشاء الطبقة المطلوبة:
```
اصنف المدينة {
    جملة الأسم ديناميكي
    جملة السكان ديناميكي

    استودع (اسم: ديناميكي، سكان: ديناميكي) {
        هذا.أسم = أسم
        هذا.سكان = سكان
    }
}
```
الكود البرمجي التالي يقوم بتعريف الواجهة "المعلومات":
```
واجهة المعلومات {
    أظهر_المعلومات()
}
```
الآن، يمكننا أن نجعل الطبقة "المدينة" تقبل الواجهة "المعلومات". هذا يتطلب منا إيجاد شكل للوظيفة `أظهر_المعلومات()`. فلنقم بذلك:
```
اصنف المدينة ينفذ المعلومات {
    جملة الأسم ديناميكي
    جملة السكان ديناميكي

    استودع (اسم: ديناميكي، سكان: ديناميكي) {
        هذا.أسم = أسم
        هذا.سكان = سكان
    }

    أظهر_المعلومات() {
        طباعة("اسم المدينة: ${هذا.أسم}")
        طباعة("عدد السكان: ${هذا.سكان}")
    }
}
```
بهذا الشكل، لا يمكننا إكمال عملية البرمجة بلغة عربية قووية ومتقنة حقا.
