# تعريف الطبقات و الواجهات 

في إنشاء البرمجة باللغة العربية نستخدم جملة مثل "أعرف طبقة باسم" لتعريف الطبقات، و "أعرف واجهة باسم" لتعريف الواجهات. كل طبقة أو واجهة يمكن أن تحتوي على متغيرات وأساليب محددة.

مثلا:

```العربية
أعرف طبقة باسم الحيوان {
  اسم المتغير نوع الحيوان
  اسم الدالة أصدر صوت {
    أرجع "الحيوان يصدر صوت"
  }
}

أعرف واجهة باسم النبات {
  اسم المتغير نوع النبات
  اسم الدالة ينمو {
    أرجع "النبات ينمو"
  }
}
```

عند استخدام الطبقة أو الواجهة، نمكن الاشارة الى اسمها مع القيمة التي نريد تعيينها للمتغيرات، أو الأساليب التي نريد تنفيذها.

```العربية
أعرف الحيوان الأسد {
  نوع الحيوان = "لاساد"
  دع أصدر صوت
}

أعرف النبات الورد {
  نوع النبات = "لوردي"
  دع ينمو
}
```

تقوم هذه الأسطر بإعداد كائن جديد من الطبقة الحيوان بالاسم "الأسد" وتعيين قيمة النوع "لاساد"، وكائن من الواجهة النبات بالاسم "لورد" وتعيين قيمة النوع "لوردي"، ثم تنفيذ الأساليب المحددة.
