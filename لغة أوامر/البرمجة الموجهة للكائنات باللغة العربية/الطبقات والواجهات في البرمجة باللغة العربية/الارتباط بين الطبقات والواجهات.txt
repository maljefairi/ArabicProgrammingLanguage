## الارتباط بين الطبقات والواجهات باللغة العربية

تتميز اللغة البرمجيه باللغة العربية بسهولة الاستخدام وفهم الجميع للمفاهيم البرمجية بطريقة مبسطة وواضحة. دعونا ننظر إلى كيف يمكننا إستخدام الطبقات والواجهات معاً في لغة البرمجة العربية.

1. **الطبقة (الصف)**: يمكن تعريف الطبقات كمجموعة من المعتبرات (الخصائص) والأوامر(الوظائف) التي يمكن أن تشترك فيها العناصر المشتقة من الطبقة. 

مثال: في هذا الكود نقوم بتعريف طبقة(صف) باسم "العامل" في الغة العربية:
```
صنف العامل {

    معتبر اسم: سلسلة
    معتبر رقم الهاتف: طويل
    معتبر الراتب: طفو
    
    دالة عرض_المعلومات() {
        اطبع("الاسم هو " + اسم)
        اطبع("رقم الهاتف هو " + رقم الهاتف)
        اطبع("الراتب هو " + الراتب)
    }
}
```

2. **الواجهة**: هي تعريف لمجموعة من الأدوات (الأوامر) التي يجب تحقيقها في الصفوف او الطبقات التي تنفذ الواجهة.

مثال: في الكود التالي تم تعريفة واجهة باسم "العمل" في اللغة العربية:
```
واجهة العمل {
    دالة بدء_العمل()
}
```
وفي الكود التالي تم تطبيق الواجهة على الطبقة (الصف):

```
صنف العامل: العمل {

    دالة بدء_العمل() {
        اطبع("العامل بدأ العمل")
    }
}
```

لذا، الطبقة(الصف) "العامل" الآن ملتزمة بتنفيذ جميع التوابع(الوظائف) في واجهة "العمل".
