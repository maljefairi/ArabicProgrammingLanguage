أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

### الكود البرمجي المقترح:

هذا مثال على طريقة بناء الطبقة في اللغة العربية:

```arabic
// طبقة فصل 
طبقة الفصل {
    خاص سلسلة اسم الفصل = "";
    
    عمومي الفصل (سلسلة اسم الفصل){
        هذا.اسم الفصل = اسم الفصل;
    }
    
    عمومي ثابتة لا تعيد سلسلة الحصول على اسم الفصل() {
        عود اسم الفصل;
    }
}
```

### مثال لتطبيق الكود البرمجي السابق:

```arabic
الفصل الفصل_الأول = جديد الفصل("الفصل الأول");
طباعة (الفصل_الأول.الحصول على اسم الفصل());
```

عند تشغيل الكود السابق، سيتم طباعة سلسلة "الفصل الأول" على الشاشة.
