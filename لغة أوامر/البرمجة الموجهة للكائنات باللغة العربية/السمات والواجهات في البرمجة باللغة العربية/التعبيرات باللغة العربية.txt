أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بناءً على السياق المقدم، يمكننا مناقشة مفهوم السمات والواجهات في البرمجة باللغة العربية. في البرمجة، السمة هي خاصية أو متغير ضمن كائن أو صف، أما الواجهة فهي أداة تستخدم لتعريف الأساليب (الدوال أو الوظائف) التي يجب أن تقدّمها الصفوف أو الأنواع التي تنفذها.

### السمات
الكود التالي يقدم تعريفاً لسمة (خاصية) باللغة العربية:

```arabic
فصل صنف موظف {

خصائص {
    سمة الاسم من نوع نص
    سمة الراتب من نوع رقم
}

طرق {
   اجراء زيادة_الراتب(قيمة من نوع رقم) {
     هذا الراتب = هذا الراتب + قيمة;
  }
}

}
```
الكود أعلاه يعرف صف "موظف" به سمتين هما "الاسم" و"الراتب" بالإضافة إلى دالة اسمها "زيادة_الراتب".

### الواجهات
الأمثلة التالية توضح كيف يمكن تعريف واجهة باللغة العربية:

```arabic
فصل واجهة محمول {
  طرق {
    دالة_بدون_قيمة_رجوع اتصال بدون_وسائط.
    دالة نسخة من نوع نص بدون_وسائط.
  }
}
```

هذا الكود يعرف واجهة تسمى "محمول" وتحتوي على دوال "اتصال" و "نسخة". أي صف يرغب في تنفيذ هذه الواجهة، يجب عليه تزويدها بتعريفات لهذه الدوال.
