أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# تعريف الواجهات باللغة العربية     
      
الواجهات باللغة العربية تُعَرَف عبر استخدام الكلمة الرئيسية "واجهة". بعدها نكتب اسم الواجهة. بعد الاسم، نكتب الدوال التي يمكن أن تكون جزء من هذه الواجهة.
     
في الأدناه، يمكننا رؤية مثال على تعريف الواجهات باللغة العربية:
      
```bash
واجهة الشكل {
  الدالة ارسم();
  الدالة احسب_المساحة();
  الدالة احسب_المحيط();
}
```
في هذه الواجهة، الأشكال هم الذين يتوجب عليهم تنفيذ هذه الدوال. مثلاً، إذا كان لدينا شكل مثل المستطيل، سيتوجب عليه تنفيذ دوال "ارسم"، "احسب_المساحة"، و"احسب_المحيط". 
وهذه حالة تنفيذ الواجهة:
     
```bash
صنف المستطيل ينفذ الشكل {
  الدالة ارسم(){
    # تنفيذ رسم المستطيل
  }
  الدالة احسب_المساحة(){
    # تنفيذ حساب مساحة المستطيل
  }
  الدالة احسب_المحيط(){
    # تنفيذ حساب محيط المستطيل
  }
}
```
هذا الصنف المستطيل ينفذ الواجهة "الشكل" وكل الدوال المعرفة فيها.
