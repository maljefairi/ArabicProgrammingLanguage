أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الكود البرمجي باللغة العربية الفصحى

يمكن تقديم النماذج المقترحة للكود البرمجي باللغة الفصحى كالأتي:

```
أنشئ صف جديد باسم عامل{
    لديه خاصية الاسم 
    لديه خاصية رقم الهاتف
}
```

في هذا الكود، قمنا بإنشاء صف جديد يسمى "عامل" به خاصيتان هما الاسم ورقم الهاتف.

```
أنشئ كائن من صف عامل باسم عامل1 {
    جعل خاصية الاسم تكون محمد 
    جعل خاصية رقم الهاتف تكون 1234567890
}
```

في هذا الكود، قمنا بإنشاء كائن جديد من الصف "عامل" باسم "عامل1" وأعطينا الخاصية "الاسم" القيمة "محمد" والخاصية "رقم الهاتف" القيمة 1234567890.

# الواجهات باللغة العربية الفصحى مع الأمثلة

الكود التالي يظهر كيفية تعريف وتطبيق الواجهات:

```
أنشئ واجهة باسم تحكم {
    تكون بها دالة اسمها قدّم
}

أنشئ صف جديد باسم موظف يطبق واجهة تحكم{
    تكون به دالة اسمها قدّم {
        أطبع فقط "الموظف محمد قدم"
    }
}
```

في الكود أعلاه، قمنا بتعريف واجهة تدعى "تحكم" وهي تحتوي على دالة تدعى "قدّم". بعد ذلك، أنشأنا صفًا جديدًا يدعى "موظف" يطبق الواجهة "تحكم" وتم تطبيق الدالة "قدّم" بشكل محدد للموظف.
