أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

الاجراءات والوظائف هي احدى مفاهيم البرمجة الأساسية حيث يمكنك التفكير في وظيفة كقطعة من البرنامج يمكننا استدعائها في أي وقت ومن أي مكان في برنامجنا. هذا يجعل الكود أكثر تنظيمًا وقابلية لإعادة الاستخدام، ويتيح لنا فصل المنطق والمعالجة في جزء من البرنامج.

في لغتنا البرمجية العربية الفصحى، يمكننا تعريف الوظيفة كالتالي:

```
الدالة جمع_الأرقام(رقم1, رقم2) {
  إرجاع رقم1 + رقم2;
}
```

وفيما يلي كيفية استدعاء الوظيفة:
```
س <- جمع_الأرقام(5, 3);
أطبع(س);
```

من الأمثلة أيضا التعامل مع المتغيرات، حيث يمكننا في لغتنا البرمجية العربية الفصحى العمل مع المتغيرات كالتالي:

تعريف متغير جديد:
```
فار عمر;
```

تعيين قيمة للمتغير:
```
عمر <- 25;
```

استدعاء القيمة المحفوظة في المتغير:
```
أطبع(عمر);
```

تغيير قيمة المتغير:
```
عمر <- 26;
أطبع(عمر);
```
