## الإجراءات والوظائف في البرمجة باللغة العربية 

في البرمجة، الإجراءات والوظائف هما مكونات أساسية تمكن المبرمجين من كتابة الأكواد بطريقة منظمة وقابلة لإعادة الاستخدام. في اللغة العربية، يمكننا استعمال الأفعال او الجمل المفيدة بشكل مباشر لتكوين هذه الإجراءات والوظائف.

لنأخذ مثلاً إنشاء وظيفة لحساب مجموع الأرقام:

هذا الأمر البرمجي يدل على كيفية كتابة الدالة في اللغة البرمجة العربية:

```Arabic code
دالة إجمالي(قائمة_الأرقام) {
     إجمالي = 0
     لكل رقم في قائمة_الأرقام {
         إجمالي = إجمالي + رقم
     }
     أرجع الإجمالي
}
```

في هذا الكود، نحن نعرف أولا وظيفة بالاسم "إجمالي" تستقبل قائمة من الأرقام. ثم في الدالة، نحن نستعرض كل الأرقام ونضيفها إلى المجموع الكلي. وفي النهاية، تعيد الدالة الجملة المحسوبة.

**مثال على تطبيق هذا الأمر البرمجي:**

```Arabic code
أرقامي = [1, 2, 3, 4, 5]
طباعة(إجمالي(أرقامي))
```

هنا، نحن نقوم بإستعمال الدالة التي أنشأناها لحساب مجموع الأرقام في القائمة باسم "أرقامي". القيمة المطبوعة من هذا الكود ستكون "15", والتي هي مجموع الأرقام من 1 إلى 5.
