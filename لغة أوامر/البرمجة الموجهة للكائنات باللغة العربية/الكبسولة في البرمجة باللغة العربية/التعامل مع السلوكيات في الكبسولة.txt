أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

الكود البرمجي:

```
كائن موظف 
{
    خاص اسم:نص;
    خاص عمر:رقم;

    دالة انشاء(اسم:نص, عمر:رقم)
    {
        هذا.اسم = اسم;
        هذا.عمر = عمر;
    }

    دالة بهاء()
    {
        ارجع "الاسم: " + هذا.اسم + " ، العمر: " + هذا.عمر;
    }
}
```
في الكود أعلاه, نقوم بالإعلان عن كائن جديد باللغة العربية يُدعى "موظف". ضمن هذا الكائن، لدينا خصائص اسم وعمر، بالإضافة إلى دالتين، الدالة الأولى هي دالة الإنشاء التي تتم عن طريق إدخال الاسم والعمر، والدالة الثانية هي دالة بهاء، وهي تقوم بعرض القيم المُدخلة للخصائص.

الان نستعرض كيفية تطبيق هذا الأمر البرمجي:

```
الموظف الاول = موظف( "محمد",30);
أكتب(الموظف الاول.بهاء());
```
في هذا الجزء، نجد أننا قمنا بإنشاء كائن جديد باستخدام الدالة المحددة سابقا وأعطيناه القيم "محمد" و "30" للحصول على الموظف الأول ، ثم نقوم بكتابة قيم هذا الموظف باستخدام دالة "بهاء". النتيجة ستكون: "الاسم: محمد, العمر: 30".
