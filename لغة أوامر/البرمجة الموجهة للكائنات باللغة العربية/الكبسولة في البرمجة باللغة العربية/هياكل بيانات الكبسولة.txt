# هياكل بيانات الكبسولة باللغة العربية الفصحى

في البرمجة باللغة العربية الفصحى، الكبسولة (Encapsulation) هي واحدة من الواجهات الرئيسية لمفهوم البرمجة الموجهة للكائنات. تنقسم هياكل بيانات الكبسولة إلى نوعين أساسيين: المتغيرات والبيانات.

## كبسولة المتغيرات

تقع تحت هذه الفئة كل البنود التي تم تعريفها في الكائن والتي يمكن الوصول إليها وتعديلها من خلال الأساليب داخل الكائن فقط.

أليك مثال لكبسولة المتغيرات باللغة العربية الفصحى:

```
كائن سيارة {
    متغير سرعة؛
    إجراء تحديث_السرعة(متغير السرعة_الجديدة) {
        سرعة = السرعة_الجديدة؛
    }
    إجراء عرض_السرعة() {
        عرض(سرعة)؛
    }
}

متغير سيارتي = جديد سيارة()؛
سيارتي.تحديث_السرعة(200)؛
سيارتي.عرض_السرعة()؛  // تعرض '200'
```

## كبسولة البيانات

تقع تحت هذه الفئة كل البنود التي تم تعريفها في الكائن والتي يمكن الوصول إليها وتعديلها من خلال الأساليب داخل الكائن فقط ولا تمكن الوصول إليها مباشرة من خلال الكائن.

أليك مثال لكبسولة البيانات باللغة العربية الفصحى:

```
كائن شخص {
    مخفي متغير اسم؛
    إجراء وضع_الاسم(متغير الاسم_الجديد) {
        اسم = الاسم_الجديد؛
    }
    إجراء الحصول_على_الاسم() {
        عودة اسم؛
    }
}

متغير أمين = جديد شخص()؛
أمين.وضع_الاسم("أمين بن زيادة")؛
عرض(أمين.الحصول_على_الاسم())؛  // تعرض 'أمين بن زيادة'
```

بهذه الطريقة، نكون قد طبقنا مفهوم الكبسولة في البرمجة باللغة العربية الفصحى.
