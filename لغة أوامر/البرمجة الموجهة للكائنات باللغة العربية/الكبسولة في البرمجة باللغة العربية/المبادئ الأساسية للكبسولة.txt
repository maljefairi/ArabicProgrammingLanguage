أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

الكود الرئيسي:

```العربية
كائن دفتر الحسابات {
   خصوصي متغير رصيد = 0;

   دالة عامة إيداع(مقدار) {
     إذا (مقدار > 0) {
       رصيد = رصيد + مقدار;
     } أخرى {
       أطبع("الرصيد لا يمكن أن يكون سالباً ");
     }
   }

   دالة عامة سحب(مقدار) {
     إذا (مقدار <= رصيد) {
       رصيد = رصيد - مقدار;
     } أخرى {
       أطبع("الرصيد غير كافي");
     }
   }

   دالة عامة تحقق_من_الرصيد() {
     أطبع("الرصيد الحالي هو: " + رصيد);
   }

  } 
 ```

 كيفية تطبيقه:
 لنقم بإنشاء كائن جديد من نوع دفتر الحسابات، ثم نقوم بعملية إيداع ثم نتحقق من الرصيد: 

```العربية
دع هيثم = دفتر الحسابات جديد;
هيثم.إيداع(500);
هيثم.تحقق_من_الرصيد();
#يجب أن يطبع: الرصيد الحالي هو: 500
```
ثم لنقم بعملية سحب ثم نتحقق من الرصيد:

```العربية
هيثم.سحب(200);
هيثم.تحقق_من_الرصيد();
#يجب أن يطبع: الرصيد الحالي هو: 300
```
كما نرى، نستخدم الكبسولة هنا لإخفاء الرصيد ومنع استخدامها في الأكواد الأخرى بشكل مباشر وبدون الإلتزام بالقواعد التي وضعناها في دوال الإيداع والسحب.
