# استثناءات الوقت الفعلي

في لغة برمجتنا العربية، يمكننا التعامل مع استثناءات الوقت الفعلي بشكل خاص. يتم القاء هذه الأنواع من الاستثناءات عندما تحدث أحداث غير متوقعة أثناء التنفيذ الحقيقي للبرنامج.

اليك الأمر المقترح للتعامل مع الاستثناءات الوقت الفعلي في هذه اللغة :

```bash
جرب { 
   جملة_برمجية 
} إلقاء (استثناء الوقت_الفعلي) { 
   جملة_الاستثناء
}
```
في هذا السياق، الجزء "جرب" يتبع برمجيا التشغيل. إذا حدث استثناء من نوع "الوقت الفعلي"، يتم التقاطه وتنفيذ الكود في مقطع "إلقاء".

الآن، دعنا نرى كيف يمكن تطبيق هذا على سياق فعلي :

```bash
جرب {
   افتح (ملف.txt)
} إلقاء (استثناء الوقت_الفعلي) {
   أطبع (تعذر فتح الملف)
}
```
في هذا السياق، نحاول فتح ملف للقراءة. إذا كان الملف غير موجود أو لا يمكن الوصول إليه، سيتم إلقاء استثناء من نوع "الوقت الفعلي" وتنفيذ الكود المحدد في القسم "إلقاء".
