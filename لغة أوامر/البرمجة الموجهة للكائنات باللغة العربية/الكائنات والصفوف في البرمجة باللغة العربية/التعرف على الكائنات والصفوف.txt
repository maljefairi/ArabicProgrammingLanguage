# الكائنات و الصفوف في البرمجة

في البرمجة بالعربية، يمكننا تعريف الكلاس (الصف) والأوبجيكت (الكائن) على النحو التالي:

```arabic
كلاس سيارة {
    نوع: متغير_أحرف
    لون: متغير_أحرف
    سرعة: متغير_أرقام

    بناء(نوع: متغير_أحرف، لون: متغير_أحرف، سرعة: متغير_أرقام) {
        هذا.نوع = نوع
        هذا.لون = لون
        هذا.سرعة = سرعة
    }

    دالة قود() {
        طباعة("أنا أقود " + هذا.نوع + " بلون " + هذا.لون + " بسرعة " + هذا.سرعة)
    }
}
```

في الكود أعلاه، تم تعريف صف يُدعى "سيارة" وفيه تم تحديد 3 خواص تمثل وصف للسيارة، وهي: النوع، اللون وسرعة السيارة. قمنا ببناء الكائن بواسطة الدالة الباني "بناء()" وأعطينا لها القيم الأولية لخواص الكائن.

وهناك تابع أخر هو "قود()" للمحاكاة سلوك السيارة وطريقة استخدامها.

ويمكن إنشاء كائن من هذا الصف على النحو التالي:

```arabic
متغير سيارتي = سيارة_جديدة("تويوتا", "أزرق", 160)
سيارتي.قود()
```

عند تنفيذ هذا الكود، ستظهر النتيجة التالية:

```arabic
أنا أقود تويوتا بلون أزرق بسرعة 160
```
