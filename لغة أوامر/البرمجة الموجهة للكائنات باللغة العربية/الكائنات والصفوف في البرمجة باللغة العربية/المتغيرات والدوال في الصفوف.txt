أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# تعريف الكائنات والصفوف

```arabic
فصل كائن مربع {
    متغير خاص طول = 0 ; 
    متغير خاص عرض = 0 ; 

    دالة عامة وسطي_المقادير( المتغير ط , المتغير ع ){
        الطول = ط ;
        العرض = ع ;
    }
}
```

في المثال الأول، تم تعريف كائن باسم "مربع". هو عبارة عن فصل فيه متغيرين خاصين "الطول" و"العرض" يتم تعيين قيم لهما من خلال الدالة "وسطي_المقادير".

# المتغيرات والدوال في الصفوف

```arabic
فصل كائن مستطيل {
    متغير خاص طول = 0 ; 
    متغير خاص عرض = 0 ; 

    دالة عامة وسطي_المقادير( المتغير الط , المتغير الع ){
        الطول = الط ; 
        العرض = الع ;
    }

    دالة عامة حسب_المساحة() {
        اعد الطول * العرض ; 
    }
}
```

في هذا المثال، قمنا بتعريف دالة داخل الفصل تحسب مساحة المستطيل باستخدام الطول والعرض. الدالة "وسطي_المقادير" تقوم بتحديث القيم، في حين أن الدالة "حسب_المساحة" تقوم بحساب المساحة.
