# الكائنات والصفوف في البرمجة باللغة العربية

في البرمجة التوجهة للكائنات، الكائنات هي العناصر الأساسية المستخدمة. كنتيجة، من الضروري تعرف كيفية إنشاء واستخدام هذه الكائنات في غالبية البيئات والتطبيقات البرمجية. الكائنات تعرف عبر الصفوف، والتي تعتبر قوالب للكائنات.

الكود البرمجي:

```
صف شخص {
    خاصة:
        اسم : نص,
        عمر : رقم
    عامة:
        دالة ضبط_الاسم(اسم_جديد : نص) {
            هذا.الاسم = اسم_جديد;
        }
        دالة الحصول_على_الاسم() : نص {
            ارجاع هذا.الاسم;
        }
 }
```
 في هذا المثال، قمنا بتعريف صف يطلق عليه "شخص"، يحتوي على خاصيتين "الاسم" و "العمر". وتم تعريف وظيفتين "ضبط_الاسم" تقوم باستقبال متغير “اسم_جديد” من النوع نص وتضبط الاسم بالاسم الجديد، و “الحصول_على_الاسم” ترجع الاسم.

# الوراثة في البرمجة التوجهة للكائنات 

الوراثة هي خاصية أساسية تتيح لنا إنشاء صف جديد يستند إلى صف موجودة "الصف الأصل".

الكود البرمجي:

```
صف موظف يرث من شخص {
    خاصة:
        راتب : رقم
    عامة:
        دالة ضبط_الراتب(راتب_جديد : رقم) {
            هذا.الراتب = راتب_جديد;
        }
        دالة الحصول_على_الراتب() : رقم {
            ارجاع هذا.الراتب;
        }
  }
```

في هذا المثال، "موظف" هو صف يرث من الصف "شخص". هذه الوراثة تعني أن "موظف" يحتوي على جميع الخصائص والوظائف التي تم تعريفها في صف "شخص". بالإضافة إلى الراتب الذي تم تعريفه خصيصا في صف "موظف". 

انشاء كائن جديد من صف “موظف” وضبط اسمه وراتبه يمكن كالتالي:

```
متغير احمد = جديد موظف()
احمد.ضبط_الاسم("احمد")
احمد.ضبط_الراتب(5000)
```
في هذه الحالة تم انشاء كائن جديد من نوع “موظف” وتم تعيين اسمه "احمد" والراتب 5000.
