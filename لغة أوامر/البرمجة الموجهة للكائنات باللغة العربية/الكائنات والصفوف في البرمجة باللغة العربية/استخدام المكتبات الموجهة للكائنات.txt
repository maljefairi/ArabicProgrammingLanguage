أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إن إنشاء لغة برمجة باللغة العربية الفصحى تتطلب تحليل دقيق وواضح. حيث أن الكائنات والأصناف هما الجوانب الأساسية للبرمجة الموجهة للكائنات. اتبع الأمثلة التالية للفهم الأفضل.

## الكائنات والأصناف

في البرمجة، الكائن هو كيان يحتوي على حالة وسلوك. الأصناف هي نوع الكائن، يمكنك التفكير فيها بمثابة نموذج أو خطة لإنشاء كائنات معينة.

مثال على كائن وصف باللغة العربية:

```
أصناف سيارة {
     حالة:
         نوع السيارة(: نوع سيارة)
         لون السيارة(: لون)
         
     سلوك:
        إدارة() {
        قائل("السيارة تقود الآن...")
      }
  }
```

مثال على استخدام الكائن:

```
كائن خمس سيارات = جديد سيارة()
خمس سيارات.إدارة()
```

## المتغيرات والدوال في الأصناف

المتغيرات في الأصناف تعتبر خصائص الأصناف، والدوال هي الطرق التي يمكن من خلالها تنفيذ وظائف معينة.

مثال على ذلك:

```
أصناف كتاب{
      خصائص:
         اسم الكتاب(: نص)
         تاريخ النشر(: تاريخ)
         
      وظائف:
        عرض اسم الكتاب(){
         قائل(اسم الكتاب)   
        }
}
```

## استخدام المكتبات الموجهة للكائنات

المكتبات هي مجموعات من الأكواد التي يمكن استعادتها واستخدامها للقيام بوظائف معينة بدون الحاجة لإعادة كتابتها من الصفر.

مثال على المكتبات:

```
 استورد المكتبة الرياضية 
 س = الرياضية.جدر(16)
 قائل س
```  

يرجى ملاحظة أن الأكواد البرمجية الموضحة أعلاه ليست حقيقية ولكن صممت فقط لتوضيح الأفكار. في بناء لغة برمجة حقيقية باللغة العربية ، سيكون هناك الكثير من التفاصيل والقواعد الحساسة التي يجب اتباعها.
