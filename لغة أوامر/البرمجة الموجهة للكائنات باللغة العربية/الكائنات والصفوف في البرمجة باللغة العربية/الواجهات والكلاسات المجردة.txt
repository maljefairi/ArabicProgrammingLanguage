## تعريف الكائنات والصفوف

في اللغة البرمجية العربية المتصورة، يمكننا تعريف صف باستخدام الكلمة المفتاحية "صف"، ومن ثم اسم الصف. مثلاً:

```العربية
صف طالب {
    // سمات الطالب
    اسم: نص;
    رقم_الطالب: رقم;
  
    // الدوال
    دالة نشر_الاسم() {
        اطبع(اسم);
    }
}
```

تم في الأعلى تعريف صف بإسم "طالب" يحتوي على سمتين "اسم" من نوع نص، و"رقم_الطالب" من نوع رقم. كما ويحتوي على دالة "نشر_الاسم" التي تقوم بطباعة الاسم.

إذا أردنا إنشاء كائن من الصف "طالب" نستخدم الكلمة المفتاحية "جديد" متبوعة بإسم الصف وقوسين:

```العربية
طالب_جديد = جديد طالب();
```

بعد ذلك، يمكننا تعيين القيم لسمات الكائن الجديد واستدعاء دواله. كمثال، لنقم بتعيين الاسم ورقم الطالب، ثم نستدعي الدالة "نشر_الاسم":

```العربية
طالب_جديد.اسم = "أحمد";
طالب_جديد.رقم_الطالب = 12345;

طالب_جديد.نشر_الاسم();  // "أحمد"
```

فيما فوق، نرى أن الدالة "نشر_الاسم" تقوم بطباعة الاسم "أحمد" الذي قمنا بتعيينه للكائن "طالب_جديد".
