أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

تتطلب البرمجة الموجهة للكائنات مفهومين أساسيين: الكائنات والصفوف. 

الكائن هو وحدة أساسية في البرمجة الموجهة للكائنات ويحتوي على حالة وسلوك. على سبيل المثال، يمكننا تصور كائن السيارة في الحياة الحقيقية بأنه يحتوي على حالة (اللون، الموديل، السرعة الحالية) وسلوك (القيادة، التوقف).

الصف هو "قالب" يتم استخدامه لإعداد الكائنات. 

في لغة البرمجة العربية، يمكن تطبيق هذا الأمر كما يلي:

الكود البرمجي باللغة العربية
```
صف سيارة {
  خاص لون ألوان؛
  خاص الموديل نص؛
  خاص السرعة الحالية صحيح؛

  دالة قيادة (السرعة صحيح) {
    السرعة الحالية = السرعة؛
  }

  دالة توقف () {
    السرعة الحالية = 0؛
  }
}

```
وهنا نقوم بإنشاء كائن جديد من الصف سيارة:

```
سيارة مايكار = جديد سيارة؛
مايكار.قيادة (60)؛
```
ملاحظة: هذا الكود البرمجي هو سياق توضيحي وقد لا يعمل بالفعل ضمن واقع بيئة برمجة معينة. يتعين عليك التحقق من صحة هذه الأكواد بناءً على اللغة البرمجية التي تعمل بها.
