# الرمز البرمجي باللغة العربية الفصحى

في اللغة البرمجية التي تستند إلى اللغة العربية الفصحى، مثلاً هيكل البرمجة الموجهة للكائنات يمكن أن يرسم بالطريقة التالية:

```javascript
كائن شخص {
    عنصر الاسم:
    عنصر العمر:

    اجراء التعريف(الاسم، العمر) {
        هذا.الاسم = الاسم;
        هذا.العمر = العمر;
    }

    اجراء عرض_التفاصيل() {
        طباعة("الاسم: " + هذا.الاسم);
        طباعة("العمر: " + هذا.العمر);
    }
}

كائن محمد = جديد شخص("محمد", 30);
محمد.عرض_التفاصيل();
```

في المثال السابق، `كائن` التعامل مع المتغير 'شخص' ككائن، وتستخدم الكلمة الرئيسية 'جديد' لإنشاء كائن جديد، واستخدمنا 'اجراء' لتعريف دالة أو طريقة في الكائن. أخيرا، استخدمنا الكلمة 'طباعة' لإظهار النتيجة.
