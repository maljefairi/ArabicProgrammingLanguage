أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الأدوات والتقنيات للبرمجة الكائنية باللغة العربية
## مكتبات الأكواد في اللغة العربية

لنفترض أنه تم تطوير لغة برمجة باللغة العربية الفصحى تُسمى "عربي كود". هذه لغة تحويلية قوية تقوم بتحويل النص باللغة العربية إلى الأكواد البرمجية بلغات مثل JavaScript. 

تخيل أنك تريد أن تكتب برنامج بسيط لإنشاء كائن شخص (Person):

مثال على كود:
```Arabic
أكتب "الرجاء إدخال اسم الشخص"
دع اسم = قراءة()
أكتب "الرجاء إدخال العمر"
دع عمر = قراءة()

دع الشخص = جديد شخص(اسم, عمر)
أكتب الشخص.طباعة()
```
أين:
* الأمر `أكتب` يُستخدم لطباعة الرسالة على شاشة الكمبيوتر.
* الأمر `قراءة()` يتيح إدخال المستخدم.
* الأمر `جديد شخص(اسم, عمر)` ينشئ كائن جديد من نوع "شخص".
* الأمر `الشخص.طباعة()` يستدعي الوظيفة "طباعة" المُعرّفة في الكائن "الشخص".

لنفترض أن الصف "شخص" تم تعريفه مسبقًا في مكتبة الأكواد على الوجه الآتي:

```Arabic
دع شخص = صف {
  وظيفة بناء(اسم, عمر) {
    هذا.اسم = اسم
    هذا.عمر = عمر
  }

  وظيفة طباعة() {
    أكتب "الاسم: " + هذا.اسم
    أكتب "العمر: " + هذا.عمر
  }
}
```

ملاحظة: الكود الفوق بيعتمد على فهم مبادئ البرمجة الكائنية ومصطلحات لغة البرمجة المطلوب تنفيذها، لكنه يبقى تقريبي لغرض التوضيح. وهذا الاقتراح يُحتاج التطوير الكامل للنظام البرمجي الذي سيقبل البرمجة باللغة العربية الفصحى وتنفيذ الجمل بشكل صحيح.
