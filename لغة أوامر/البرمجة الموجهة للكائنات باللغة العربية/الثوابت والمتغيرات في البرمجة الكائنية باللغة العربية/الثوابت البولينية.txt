# الثوابت البولينية
الثوابت البولينية هي نوع من الثوابت التي يمكن أن تحمل قيمتين فقط: صحيح أو خطأ. في البرمجة باللغة العربية، يمكن تعريف الثابتة البولينية بالشكل التالي:

```arabic
الثابت البوليني طالب_ناجح = صحيح;
```

في هذا المثال، أعلنا الثابت البوليني باسم "طالب_ناجح" وأعطيناه القيمة "صحيح".

إذا كنا نريد أن نعرف إذا كان الطالب ناجحًا أم لا، يمكننا استخدام الكود التالي:

```arabic
إذا (طالب_ناجح = صحيح) { 
   طباعة("الطالب ناجح"); 
} وإلا {
   طباعة("الطالب غير ناجح");
}
```

في هذا المثال، نستخدم الثابت البوليني "طالب_ناجح" في جملة برمجية تحكمية (إذا...وإلا). إذا كان الطالب ناجحاً (أي، إذا كان الثابت البوليني "طالب_ناجح" يحمل القيمة "صحيح")، وقتها سيطبع البرنامج "الطالب ناجح". وإلا، سيطبع "الطالب غير ناجح".
