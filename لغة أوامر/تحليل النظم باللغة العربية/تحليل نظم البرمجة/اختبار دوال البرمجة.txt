أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الكود البرمجي باللغة العربية

دعونا نعتبر الكود البرمجي البسيط التالي في لغة الجافا ولنقم بتحويله إلى اللغة العربية:

```
public class برمجة 
{
   public static void main(String []args) 
   {
      System.out.println("مرحبا بك في البرمجة باللغة العربية!");
   }
}
```
