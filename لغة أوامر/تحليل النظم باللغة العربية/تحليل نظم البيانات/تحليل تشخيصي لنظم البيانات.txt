أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

فكرة البرمجة باللغة العربية هي أن يتمكن الأشخاص من التعبير عن الأفكار البرمجية باللغة العربية دون الحاجة للإقناع بتعلم لغة إنجليزية أو أي لغة أخرى.

لنفترض أننا نريد كتابة برنامج بسيط لطباعة رسالة "مرحبا بكم في عالم البرمجة باللغة العربية" على الشاشة. في لغة برمجة بالإنجليزية مثل الجافا، قد يكون الكود كما يلي:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("مرحبا بكم في عالم البرمجة باللغة العربية");
    }
}
```
لكن، في لغتنا التجريبية، قد نشكل الجملة بشكل مختلف وعلى النحو الآتي:

```arabic
البرنامج الرئيسي هو {
    الدالة الرئيسية تقوم بـ {
        أكتب على الشاشة "مرحبا بكم في عالم البرمجة باللغة العربية";
    }
}
```
قمنا بتكوين الجملة بطريقة فيها مزيد من المرونة والقرب للغة العربية الفصحى، وأصبحت أكثر وضوحا وقابلية للفهم، حيث ذكرنا "البرنامج الرئيسي" بدلا من "class Main"، و "الدالة الرئيسية" بدلا من "public static void main". ثم قمنا بالإشارة إلى "أكتب على الشاشة" بدلا من "System.out.println".
