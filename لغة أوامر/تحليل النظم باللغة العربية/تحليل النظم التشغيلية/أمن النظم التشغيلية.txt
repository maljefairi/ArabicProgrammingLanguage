أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إذا كنا ننشئ لغة برمجة باللغة العربية الفصحى، قد تكون هكذا:

إعتبر الكود البرمجي التالي، يعيّن متغيّر بإسم "عدد" ويعطيه القيمة 5، ثم يطبع هذه القيمة:

```
# الكود باللغة العربية
عرّف عدد = 5
اطبع(عدد)
```

هذا الكود بلغة البرمجة العربية فصحى، يتم ترجمته إلى الإنجليزية بالشكل التالي:

```
# The equivalent code in Python
number = 5
print(number)
```

الأمر "عرّف" يتم استخدامه لتعريف متغير جديد، بينما الأمر "اطبع" يتم استخدامه لطباعة قيمة المتغير.

بالمثل، إذا كان لدينا دالة تحسب المجموع، قد يكون الكود باللغة العربية كما يلي:

```
# الكود باللغة العربية
عرّف دالة مجموع(رقم1، رقم2):
    عيد الرجوع رقم1 + رقم2

اطبع(مجموع(3، 2))
```

وهذا ما يوازي الكود البرمجي بلغة البايثون:

```
# The equivalent code in Python
def sum(number1, number2):
    return number1 + number2

print(sum(3, 2))
```

الأمر "عرّف دالة" يتم استخدامه لتعريف دالة جديدة، و "عيد الرجوع" بالعربية يعني "return" بالإنجليزية والتي تستخدم للرجوع بقيمة من الدالة.
