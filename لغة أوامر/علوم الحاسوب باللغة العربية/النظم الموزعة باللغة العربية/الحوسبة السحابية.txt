أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

### إنشاء محرك بحث بسيط للبيانات في النظم الموزعة بالاستخدام لغة البرمجة باللغة العربية 

1. تحديد البيانات:
    ```
    البيانات = "هذا الكتاب من أفضل الكتب الذي قرأته"
    ```
2. تصميم الدالة لتقسيم البيانات إلى كلمات:
    ```
    دالة تقسيم(نص):
        إرجاع نص.قسم بحسب(" ")
    ```

3. استدعاء الدالة للقيام بفصل الكلمات وتخزينها في متغير: 
    ```
    الكلمات = تقسيم(البيانات)
    ```
4. تصميم الدالة للبحث عن كلمة:
    ```
    دالة البحث(الكلمة، اللائحة): 
        إذا الكلمة في اللائحة:
            إرجاع صحيح 
        وإلا: 
            إرجاع خاطئ 
    ```

5. استدعاء الدالة للبحث عن كلمة معينة في البيانات: 
    ```
    عملية البحث = البحث("الكتاب", الكلمات) 
    أطبع(عملية البحث)
    ```
إذا كانت الكلمة موجودة في اللائحة، سيتم طباعة 'صحيح'. 

### ملاحظة: هذا برنامج باللغة العربية الفصحى، لن يعمل في معظم البيئات التقليدية للبرمجة. يحتاج هذا البرنامج إلى مترجم خاص بالغة البرمجة باللغة العربية.
