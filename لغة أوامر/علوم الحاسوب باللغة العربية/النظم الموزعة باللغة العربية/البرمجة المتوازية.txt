# البرمجة المتوازية باللغة العربية

سنتطرق لكتابة بعض الجمل البرمجية بلغة برمجة نسخة من الجافا العربية، الهدف هو إعلامك بطريقة التعبير عن الأوامر البرمجية باللغة العربية الفصحى.

## الكود البرمجي في الجافا العربية:

```java
عامة فئة البرنامج_الرئيسي {
    عامة ثابتة خالية من القيمة الدالة الرئيسية(جدول من السطور الدالة الرئيسية_النصي[]) جرى {

	تمهيد تنفيذ = مدير_التنفيذ.جديد العامل_اشغال_ثابتة();

	للكل(صحيح i = 0، i أصغر من 10، i += 1) {
	    تنفيذ.تشغيل(جديد المهمة_متوازية(i));
	}

	تنفيذ.إغلق();
    }
}

عامة فئة المهمة_متوازية ينفذ قابل_للتشغيل {
    خاص آخر من النوع صحيح رقم;

    عامة المهمة_متوازية(صحيح رقم) {
	هذا.رقم = رقم;
    }

    ثابتة خالية_من_القيمة الدالة تشغيل() جرى {
	نظام.خرج.طباعة("المهمة " + هذا.رقم + " تنفذ الآن.");
    }
}
```

## مثال للتطبيق:

سيعرض هذا البرنامج نتائج الاداء الخاص بعملية المعالجة المتوازية عن طريق اعلامك بالمهمة التي تتم الآن ورقمها. ولكن اعلم ان النتائج قد تختلف بناءا على عدد الذي تم إدخاله ومدى سرعة وأداء وحدة المعالجة المركزية.
