# البرمجة باللغة العربية الفصحى 

في هذا المثال، سنكتب بعض الأكواد البسيطة باللغة العربية الفصحى. هناك العديد من اللغات التي تدعم البرمجة باللغة العربية مثل لغة "كلم" او " عرب " التي تتيح كتابة الكود باللغة العربية. لكن سنتناول الأمثلة البسيطة في اللغة الانكليزية وترجمتها للغة العربية بشكل صحيح ومقبول.

## مثال على التحليل الأمني / الجدران النارية باللغة العربية

مثلا نأخذ الأمر البسيط الإنشاء المتغير، في لغة برمجة مثل Python كالتالي:

```python
x = 10
print("العدد هو:", x)
```
نستطيع ترجمته باللغة العربية الى

```python
متغير دالة_الطباعة
دالة_الطباعة = 100
اطبع ("العدد هو:", دالة_الطباعة)
```

مثال آخر لكتابة جدار ناري بسيط، في لغة برمجة مثل Java:

```java
public class Firewall {
    private boolean isActive;

    public Firewall(boolean isActive) {
        this.isActive = isActive;
    }

    public void blockPort(int port) {
        System.out.println("Blocked port: " + port);
    }
}
```

يستطيع ان يترجم الى اللغة العربية بشكل مباشر الى
```java
عام فصل الجدار_الناري {
    خاص بوولين تفعيل;

    عام الجدار_الناري(بوولين تفعيل) {
        هذا.تفعيل = تفعيل;
    }

    عام غفلابلوكبورتبوك (عدد مرتبة) {
        طباعة_بالنظام.خط_طباعة("منع تصفح منفذ: " + مرتبة);
    }
}
```
الجدير بالذكر ان البرمجة باللغة العربية قد تكون صعبة لمن يستخدم غير اللغة العربية كلغته الأم وذلك يتوقف على البرمجيات المستخدمة وتوافقها مع اللغة العربية وكذلك قد يواجه البرمج الناطق باللغة العربية بعض الصعوبات في البرمجة بالعربية وذلك لعدم توافر الكثير من المراجع باللغة العربية.
