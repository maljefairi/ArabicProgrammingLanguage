أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

**1- عبارة برمجية لاختبار عدد:**

``` عربي
اذا(العدد > 5) {
    أطبع("العدد أكبر من خمسة");
} والا {
    أطبع("العدد ليس أكبر من خمسة");
}
```

هنا، يتم اختبار ما إذا كان العدد هو أكبر من 5، وإذا كان كذلك، سيتم الطباعة "العدد أكبر من خمسة"، وإذا لم يكن كذلك، سيتم الطباعة "العدد ليس أكبر من خمسة".

**2- عبارة برمجية للتكرار (loop):**

``` عربي
لكل(عدد في الأعداد) {
    أطبع(العدد);
}
```

هنا، هذا البرنامج الصغير يتكون من حلقة تكرار تطبع كل عدد ضمن قائمة الأعداد.

**3- عبارة برمجية لتعريف دالة:**

``` عربي
الدالة جمع(العدد_الأول، العدد_الثاني) {
    عودة العدد_الأول + العدد_الثاني;
}
```
هنا، تم تعريف دالة تدعى "جمع" تقوم بإرجاع نتيجة جمع العدد الأول والثاني. 

يرجى ملاحظة أن هذه مجرد مثال تمامًا وقد يتغير الكود حسب اللغة البرمجية العربية التي تم استخدامها.
