أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

```markdown
# إنشاء لغة برمجة باللغة العربية الفصحى، نستطيع استعمال الجمل الصحيحة مثل:

## جملة الطباعة
لكتابة "مرحبا بك في البرمجة بالعربية" نستخدم الأمر التالي:

```
اكتب("مرحبا بك في البرمجة بالعربية")
```

## الدالات
لتعريف دالة تقوم بجمع رقمين، نستخدم الجملة التالية:

```
دالة جمع (العدد الأول، العدد الثاني):
    جمع = العدد الأول + العدد الثاني
    ارجع جمع
```

بعد تعريف الدالة، يمكننا استخدامها كالتالي:

```
اكتب(جمع(3, 5))
```

هذا البرنامج سيقوم بطباعة "8" في النهاية اذا تم تنفيذه.

## الأوامر الشرطية
للتحكم في تدفق البرنامج، نستخدم الأوامر الشرطية، مثل الجملة التالية التي تقوم بالتحقق إذا كان العدد زوجي أو فردي:

```
دالة الزوج_الفردي(العدد):
    إذا العدد مقسوما على 2 يكون مساوي لصفر:
        اكتب ("العدد " + العدد + " هو زوجي")
    وإلا:
        اكتب ("العدد " + العدد + " هو فردي")
```

نستخدم هذه الدالة كالأتي:

```
الزوج_الفردي(7)
```

هذا البرنامج سيقوم بطباعة "العدد 7 هو فردي" إذا تم تنفيذه.
```
