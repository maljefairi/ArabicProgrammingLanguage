# البرمجة باللغة العربية الفصحى
تتوفر هنا بعض الكلمات والجمل البرمجية باللغة العربية الفصحى.

1. قائمة الوسائط:
   ```
   القائمة اسماء = [ "محمد", "علي", "فاطمة", "زينب" ]
   ```

2. الجمل الشرطية:
   ```
   إذا  العدد > 10 فقل "كبير"
   وإلا فقل "صغير"
   ```

3. الحلقات الدورية:
   ```
   من_أجل كل اسم في اسماء فقل "مرحباً", اسم
   ```

4. الدوال أو الوظائف:
   ```
   دالة جمع (أ, ب):
       جواب = أ + ب
       أعد الجواب
   ```

5. الكائنات والخصائص والأساليب:
   ```
   الكائن طالب:
       الخاصية الاسم, العمر
       
       الدالة تعريف(اسم, عمر):
           الاسم = اسم
           العمر = عمر
       
       الدالة نشر_المعلومات():
           قل "الاسم :", الاسم, "و العمر :", العمر

   طالب1 = طالب.تعريف("حسين", 21)
   طالب1.نشر_المعلومات()
   ```

إليك مثال على كود برمجي بسيط باللغة العربية:

```python
دالة جمع (أ, ب):
   جواب = أ + ب
   أعد الجواب

العدد_أ = 10
العدد_ب = 20

نتيجة = جمع(العدد_أ , العدد_ب)
قل "نتيجة الجمع هي", نتيجة
```

يرجى ملاحظة أن الكود المقترح هو فقط لأغراض التوضيح وقد لا يعمل في بيئة التنفيذ الحقيقية.
