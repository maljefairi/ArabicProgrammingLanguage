أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

**البرمجة باللغة العربية**

لنأخذ لغة البرمجة "فارسكربت" Farscrip كمثال على لغة برمجة باللغة العربية، حيث أنها واحدة من اللغات التي تم تطويرها لتكون باللغة العربية.

الكود البرمجي:

"إذا" و "وإلا" للشروط:

```farscript
متغير أ = ١٠
متغير ب = ٢٠

إذا(أ < ب) 
{    
    أكتب("ب هو الأكبر") 
} 
وإلا 
{   
    أكتب("أ هو الأكبر") 
}
```

"لكل" للتكرار:

```farscript
لكل(متغير i من ١ إلى ١٠)
{
     أكتب(i)
}
```

لتعريف الدوال أو التوابع نستخدم "دالة":

```farscript
دالة الجمع(متغير أ، متغير ب) 
{  
    أكتب( أ + ب ) 
}
```
بالنسبة للطرق الأخرى أو المهام الأخرى في البرمجة، يمكن تطبيق نفس الطريقة، وإنشاء الأوامر مثل تعريف المتغيرات، الدوال، وشروط الإذا وإلا، وغرف التكرار مثل "لكل" و"بينما" بشكل مشابه.
