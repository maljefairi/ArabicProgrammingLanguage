أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

الكود البرمجي العربي يمكن أن يكون شيئًا مثل:

```arabic
أذا كان (درجة >= 90) {
    اطبع("العلامة النهائية هي: أ")
} وإلا أذا كان (درجة >= 80) {
    اطبع("العلامة النهائية هي: ب")
} وإلا أذا كان (درجة >= 70) {
    اطبع("العلامة النهائية هي: c")
} وإلا أذا كان (درجة >= 60) {
    اطبع("العلامة النهائية هي: D")
} وإلا {
    اطبع("العلامة النهائية هي: F")
}
```

وهذا البرنامج ببساطة يأخذ متغير يُسمى "درجة" ويقارنه بعدة قيم. يطبع البرنامج عبارات معينة بناءً على القيمة التي يحتويها المتغير.

مثال آخر يمكن أن يكون على شكل دالة بسيطة تقوم بحساب الراتب النهائي بناء على عدد الساعات العمل والأجر للساعة الواحدة.
```arabic
دالة احسب_الراتب(ساعات العمل, أجر_الساعة){
     ارجع ساعات_العمل * أجر_الساعة
}
```

في هذه الوظيفة، تقوم الدالة "احسب_الراتب" بأخذ عدد ساعات العمل و أجر_الساعة كمدخلات, تضربهم معاً , وترجع النتيجة, يمكن استخدامها كما يلي:

```arabic
اُكتب(احسب_الراتب (40, 15))
```

هذا البرنامج سيطبع الراتب للشخص الذي يعمل 40 ساعة في الأسبوع بمعدل أجر يبلغ 15 للساعة الواحدة.
