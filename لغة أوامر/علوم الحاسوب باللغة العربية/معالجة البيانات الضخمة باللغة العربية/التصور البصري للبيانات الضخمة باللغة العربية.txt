أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الكود البرمجي باللغة العربية الفصحى 
## الدالة
```arabic
دالة حسب_مجموع(الأعداد) 
تبدأ
  مجموع = صفر 
  لكل رقم في الأعداد
  تبدأ
     مجموع = مجموع + رقم
  تنتهي
  أرجع مجموع
تنتهي
```
## الدوال الشرطية
```arabic
إذا (حالة) تبدأ
  # أوامر لتنفيذها في حال كانت الشرطية صحيحة
تنتهي
وإلا تبدأ
  # أوامر لتنفيذها في حال كانت الشرطية خاطئة
تنتهي
```
## حلقات التكرار
```arabic
بينما (شرط) تبدأ
  # كود لتنفيذه طالما الشرط مستوفى
تنتهي
```
# مثال
```arabic
دالة حسب_مجموع(الأعداد) 
تبدأ
  مجموع = صفر 
  لكل رقم في الأعداد
  تبدأ
     مجموع = مجموع + رقم
  تنتهي
  أرجع مجموع
تنتهي

# تطبيق الدالة
أعداد = [1, 2, 3, 4, 5]
اكتب خروج(حسب_مجموع(أعداد))
```
في المثال السابق، أنشأنا دالة لحساب مجموع الأعداد في قائمة ومن ثم استدعيناها باستخدام قائمة من الأعداد. الناتج المتوقع لهذا الكود سيكون 15.
