أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بالجملة العربية الفصحى، من الممكن أن يظهر البرمجة بغية توضيح الطريقة الفعلية للترجمة من العربية إلى الكود. 

فلنأخذ مثال على بناء دالة بسيطة لجمع رقمين:

بدلاً من كتابة الكود بهذه الطريقة بالانجليزية:
```python
def add(x, y):
 return x + y
```
سنكتبه بالعربية فصحى كالتالي:
```python
دالة أضف(عدد1, عدد2):
  أعد عدد1 + عدد2
```

ونستطيع دعوة هذه الدالة بهذه الطريقة بالهتمام بالتعليمات البرمجية الاصلية هكذا:

بالانجليزية:
```python
result = add(3, 4)
print(result)
```

بالعربية الفصحى:
```python
النتيجة = أضف(3, 4)
اطبع(النتيجة)
```

بتطبيق هذه الطريقة، يصبح الكود برمجي مقروء ومفهوم حتى لغير البرمجيين. نقطة مهمة أخرى هو أن المتغيرات والدوال يجب أن تسمى بطريقة توضح وظيفتها.
