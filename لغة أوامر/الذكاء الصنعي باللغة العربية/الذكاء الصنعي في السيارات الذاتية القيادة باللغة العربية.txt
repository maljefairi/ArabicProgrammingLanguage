أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

من أجل إنشاء لغة برمجة باللغة العربية الفصحى، سنحتاج إلى تحديد الجمل والأمر البرمجي الذي نريد استخدامه. فلننظر إلى المثال التالي :

أمر رئيسي مثل إنشاء متغير قد يكون كما يلي :

```
أذكر عدد اسمه سنة_التخرج وقيمته 2020
```
والذي يعادل فى لغة برمجة JavaScript مثلا :

```javascript
let graduation_year = 2020;
```

و أيضا كيف نطبق الجمل الشرطية على نفس السياق، قد يكون هكذا :

```
إذا كانت سنة_التخرج تساوي 2020 فقل "تخرجت في الوقت المحدد"
أو غير ذلك قل "لم تتخرج في الوقت المحدد"
```
والذي يعادل في لغة برمجة JavaScript :

```javascript
if (graduation_year == 2020){
    console.log("Graduated on time");
} else {
    console.log("Did not graduate on time");
}
```

الكود البرمجي باللغة العربية نحتاج لإنشاء مكتبة خاصة لتفسيره وتحويله للغة برمجة معترف بها ك JavaScript مع اتباع القواعد النحوية والإملائية العربية بطريقة صحيحة.
