أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

```markdown
## الكود البرمجي باللغة العربية الفصحى

**ملاحظة**: قائمة الأكواد التالية في لغة البرمجة باللغة العربية الفصحى ليست من واقع حقيقى بل هي فقط للاستعراض ولا تعمل بالتأكيد.

دعونا نفترض أننا نعمل في لغة برمجة تحمل اسم "أرابيكو" مثلا. الأمثلة التالية توضح الطريقة المقترحة لكتابة الكود باللغة العربية الفصحى.

### متغيرات

تعريف متغير يعمل كالتالي:

```
جعل اسم <المتغير> هو <القيمة>
```
مثال:
```
جعل اسم الجملة هو "مرحباً بك في العالم البرمجي"
```
هذا يعني أننا قمنا بإعداد المتغير `الجملة` بالقيمة "مرحباً بك في العالم البرمجي".

### الدوال

إليك كيفية تعريف دالة:

```
وظيفة <اسم الدالة> هو <الكود>
```
مثال:
```
وظيفة رسالة الفحص هو
أظهر "النظام قيد العمل بشكل صحيح"
```
هذا يعني أننا قمنا بتعريف وظيفة التحقق وأنها ستظهر رسالة "النظام قيد العمل بشكل صحيح" عند استدعائها.

### شروط

إليك كيفية استخدام الشروط:

```
إذا <الشرط> ثم
    <الإجراء>
وإلا
    <الإجراء>
```
مثال:
```
إذا الجملة تساوي "مرحباً" ثم
    أظهر "مرحباً بك!"
وإلا
    أظهر "أنا لا أفهم"
```
هذا يعني إذا كانت قيمة الجملة مساوية لـ "مرحباً" ، سيتم عرض "مرحباً بك!" ، وإلا ستظهر الرسالة "أنا لا أفهم".
```
