أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

مرحبا، اللغة البرمجية المقترحة تحتوي على الكلمات الأساسية الموجودة في لغات البرمجة المعروفة ولكنها مترجمة إلى اللغة العربية الفصحى.

فالجمل البرمجية التالية هي مثال على تعريف متغير وتعيين قيمة له في اللغة العربية المقترحة:
```بالعربية
اكتب الحقيقة عدد = ١٠
```
هذه الجملة تعادل:
```python
float number = 10.0
```
وهذا مثال آخر للدالة التي تقوم بطباعة النص على الشاشة:
```بالعربية
اكتب الدالة بنر(نص)
   نطق(نص)
نهاية الدالة
```
تعادل هذه الجملة:
```python
def print(text):
   print(text)
end function
```
وهذا مثال على كيفية استخدام هذه الدالة:
```بالعربية
بنر("مرحبا بك في اللغة البرمجية العربية!")
```
وهذه الخطوات تعادل الأمثلة السابقة ولكن باللغة الإنجليزية:
```python
print("Welcome to arabic programming language!")
```
هذه ليست لغة برمجية حقيقية ولكنها مثال على كيف يمكن ان تبدو لغة برمجية باللغة العربية الفصحى. قد يحتاج الأمر إلى مزيد من البرمجة لتحقيق هذا الاقتراح في الواقع.
