أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إليك مثال على وظيفة بسيطة في لغة برمجة باللغة العربية. سأستعمل الترجمة التقليدية لبعض الكلمات الشائعة في البرمجة:

```arabic
دالة جمع(العدد_الاول, العدد_الثاني) {
    ارجع العدد_الاول + العدد_الثاني;
}
```

ترجمة هذا الكود إلى الإنجليزية ستكون كالأتي:

```english
function add(first_number, second_number) {
    return first_number + second_number;
}
```

هنا، نقوم بتعريف دالة تقوم بجمع رقمين ببساطة. استخدمنا كلمة "دالة" لترجمة `function`، وكلمة "ارجع" لترجمة `return`.

مثال على الاستخدام في البرمجة العربية:

```arabic
أكتب(جمع(5, 3));
```

سيكون النتيجة هي `8`. هذا المثال يستعمل الدالة "أكتب" لطباعة النتيجة، وهي مرادفة ل `print` بالإنجليزية.
