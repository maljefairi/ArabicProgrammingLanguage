أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## بناء لغة برمجة باللغة العربية الفصحى

هذه لغة برمجة تجريبية باللغة العربية:

```arabic
بداية
    أعرف س = 5
    إذا س > 3 
        فأكتب "الرقم أكبر من 3"
    وإلا 
        فأكتب "الرقم أصغر من 3"
نهاية
```
في حين يكون الكود البرمجي بلغة برمجة مشهورة مثل الجافا كالتالي:

```java
public class Main {
    public static void main(String[] args) {
        int s = 5;
        if (s > 3) {
            System.out.println("الرقم أكبر من 3");
        } else {
            System.out.println("الرقم أصغر من 3");
        }
    }
}
```

أضفنا عنصر القراءة الطبيعية في اللغة العربية بجعل المفاتيح كلمات قراءة في العربية مثل "بداية"، "نهاية"، "أعرف" و "أكتب".

## بالأمن السيبراني باللغة العربية

الأمن السيبراني هو حماية الأنظمة الحاسوبية والشبكات والبرامج من الهجمات الرقمية. من أجل بناء الأمن السيبراني في لغتنا البرمجية، سنضيف كود البرمجة التالي:

```arabic
بداية
    أعرف كلمة_مرور = "123456"
    أعرف محاولة_الوصول = "654321"
    إذا كلمة_مرور == محاولة_الوصول 
        فأكتب "الوصول مُتاح"
    وإلا 
        فأكتب "الوصول مُرفوض"
نهاية
```

هذا الكود البرمجي عبارة عن نظام بسيط للتحقق من كلمة السر. يتم تخزين كلمة المرور ومحاولة الوصول، ومن ثم يتم التحقق من ما إذا كانتا متطابقتين. إذا كانتا متطابقتين، يتم السماح بالوصول. وإلا، يتم رفض الوصول.

يجب أن نلاحظ أن هذا لا يعتبر أمان سيبراني حقيقيَ، بل انه مجرد طريقة بسيطة للتحقق من المطابقة. في العالم الحقيقي, الأمان السيبراني أكثر تعقيدًا بكثير ويشمل التشفير، الشهادات، الهاش، السلطات الموثوقة، وأنواع الهجمات السيبرانية، والوقاية منها الخ.
