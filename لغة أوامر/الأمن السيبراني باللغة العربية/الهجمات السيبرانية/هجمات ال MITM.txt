# اللغة البرمجة باللغة العربية الفصحى
تعد البرمجة باللغة العربية أسلوبًا محتملًا لزيادة الوصول والفهم في بيئات البرمجة. في الأمان السيبراني، يمكننا تطوير كود للتعامل مع هجمات من نوع "الرجل في الوسط" (MITM). في هذا المثال، سنستخدم جملة بالعربية الفصحى تصف عملية الكود.

```خلطات
أعرف دالة باسم "افحص_الشبكة" تاخذ "الشبكة" كدخول{
    إذا كان "الشبكة" فارغة {
        أرجع "الشبكة ليست موجودة"
    }
    أعرف "الأمان" كقيمة أصلية صفر
    لكل "جهاز" في "الشبكة" {
        إذا كان "جهاز" به هجوم "MITM" {
            ازدد "الأمان" بواحد
        }
    }
    إذا كان "الأمان" أكبر من صفر {
        أرجع "هناك تهديد"
    } 
    أرجع "الشبكة آمنة"
}
```

في حالة وجود تهديد في الشبكة، سيقوم البرنامج بتحليل الشبكة وإعطاء تنبيه بوجود هجوم من نوع "الرجل في الوسط". بوجود هذا النوع من التحليلات الأمنية، يمكن توعية المستخدمين بالتهديدات المحتملة وتجنبها.
