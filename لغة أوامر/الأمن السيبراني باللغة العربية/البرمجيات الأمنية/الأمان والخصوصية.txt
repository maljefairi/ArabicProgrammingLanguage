أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

مرحبا، للأسف برمجة اللغة العربية ليست مشهورة وغير موجودة. لكن بإمكاننا استخدام Scratch الذي يدعم العربية، ولنتخيل أن لدينا لغة برمجة عربية بحتة لإلقاء الضوء على أمثلة رمزية قائمة على الجمل العربية الصحيحة.

### تعريف دالة

باللغة الإنجليزية سنكتب هذا الكود لتعريف دالة:

```
function greetUser(name) {
  console.log("Hello, " + name);
}
```

باللغة العربية الفصحى، هذا الأمر البرمجي قد يكون هكذا:

```
عرّف دالة تحية_للمستخدم(اسم) {
  اكتب_في_السطر("مرحبا, " + اسم);
}
```

### استدعاء دالة

بعد ذلك، نستدعي الدالة بهذا الكود باللغة الإنجليزية:

```
greetUser("Ali");
```

وباللغة العربية الفصحى، يمكن التطبيق بالشكل التالي:

```
تحية_للمستخدم("علي");
```

وهذا كود بسيط يوضح كيف يمكننا استخدام الجمل الصحيحة في البرمجة باللغة العربية. بالطبع، هذا الكود ليس ذا فائدة حقيقية إلا كمثال على تطبيق اللغة العربية في البرمجة.
