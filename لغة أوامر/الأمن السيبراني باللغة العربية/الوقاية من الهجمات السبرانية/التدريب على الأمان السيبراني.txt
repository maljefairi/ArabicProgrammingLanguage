# الأمن السيبراني باللغة العربية

صممت هذه اللغة البرمجية العربية الفصحى من أجل تسهيل مهام الأمن السيبراني ومن الممكن استخدام الأوامر المبينة أدناه:

## إنشاء قواعد الجدار الناري
``` 
أنشئ_جدار = الجدار_الناري( ثقوب=[
    الثقب(رقم="80", بروتوكول=بروتوكولات.HTTP),
    الثقب(رقم="443", بروتوكول=بروتوكولات.HTTPS)
])
```
في هذا المثال، يتم إنشاء قواعد جدار الناري للسماح برسالة خدمات HTTP و HTTPS فقط.


## التحقق من طريقة التشفير

```
جدول_التشفير = تشفير("مرحبا بكم");
اطبع(جدول_التشفير);
```
يتم تشفير النص "مرحبا بكم" بالاستعانة بالدالة التشفير وطباعة النص المشفر.

## الحماية من البرامج الضارة

```
جدول_الفحص = فحص_الفيروسات(ملف="نظام_الملفات");
اطبع(جدول_الفحص);
```
تتمكن هذه الوظيفة من فحص نظام الملفات عن البرامج الضارة وتطبع نتائج الفحص. 

يُرجى الانتباه أن هذه الأمثلة تعتبر أمثلة سياسية بين العديد من الاستخدامات الممكنة الأخرى باستخدام هذه اللغة البرمجية.
