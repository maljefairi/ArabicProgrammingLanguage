# كود برمجي عربي للأمن السيبراني

في الوقت الحالي، الأكواد البرمجية المتاحة باللغة الإنجليزية في الغالب، ولكن على سبيل المثال، يمكننا كتابة وظيفة تشفير باللغة العربية الفصحى تتكون من الرموز الأساسية فقط:

```python
# الدالة لتشفير النص
وظيفة تشفير(نص):
  مخرجات = ''
  
  ل كل حرف في نص:
    كود_حرف = ord(حرف) + 1 # رقم الحرف المقابل في جدول ASCII
    مخرجات += chr(كود_حرف) # تحويل رمز ASCII إلى حرف

  رجوع مخرجات
```

وظيفة التشفير هذه تأخذ نصاً كإدخال وتعيد نصاً تم تشفيره باستخدام التحويل البسيط - ببساطة، تزيد من قيمة ASCII لكل حرف بواحد.

مثال لتطبيق وظيفة التشفير: 

```python
نص = 'أهلاً وسهلاً'
نص_مشفر = تشفير(نص)
طباعة(نص_مشفر) # الناتج: 'اوـﺏأ ﺫـﺏأ'
```

تذكر دائما، يجب استخدام الأمان السيبراني والتشفير لحماية البيانات والمعلومات، ولكن لا ينبغي استخدامها لأغراض غير قانونية.
