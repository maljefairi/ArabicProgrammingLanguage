أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# **أمن الإنترنت**

لضمان أمن الإنترنت، يمكننا استخدام الكود البرمجي التالي باللغة العربية:

```
وظيفة تحقق_من_الأمان(الرابط) {
    إذا (الرابط.أبدأب(https)) {
        أرجع "الرابط آمن";
    } آخرا {
        أرجع "الرابط غير آمن";
    }
}
```
هذا الكود البرمجي يفحص إذا كان الرابط يبدأ بـ "https"، وهو نوع من البروتوكولات التي تضمن الأمان بين المستخدم والموقع الإلكتروني. إذا كان الرابط يبدأ بـ "https"، يقوم الكود بإرجاع "الرابط آمن". إذا لم يكن الرابط يبدأ بـ "https"، يقوم الكود بإرجاع "الرابط غير آمن".

# **مثال على تطبيق الأمر البرمجي**

للتحقق إذا كان رابط معين آمن أم لا، يمكننا استدعاء الوظيفة "تحقق_من_الأمان" وإعطائها الرابط كمدخل:

```
نتيجة = تحقق_من_الأمان("https://example.com");
أطبع(نتيجة);
```

إذا كان الرابط "https://example.com" يبدأ بـ "https"، سيكون الإخراج "الرابط آمن". إذا لم يبدأ الرابط بـ "https"، سيكون الإخراج "الرابط غير آمن".
