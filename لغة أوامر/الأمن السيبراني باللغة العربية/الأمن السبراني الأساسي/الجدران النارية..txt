أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الشيفرة البرمجية باللغة العربية
سيكون ممكن إنشاء لغة برمجة باللغة العربية الفصحى مع استخدام الجمل الصحيحة. يمكننا استعراض كود برمجي بسيط يمثل إعداد الجدران النارية في سياق الأمن السيبراني.

```arabic
عيّن "الجدار_الناري" كأمان(الشبكة) {
ادا "الشبكة" غير مأمونة {
        رسالة("الشبكة غير مأمونة، يتعذّر الاتصال")
        إيقاف التشغيل
    } وإلا {
        رسالة("الشبكة مأمونة، يمكن الاستمرار في الاتصال ")
    }
}

```

في هذا الكود البسيط، نبدء بتعيين "الجدار_الناري" كدالة لتأمين الشبكة. بعد ذلك، نتحقق إذا كانت الشبكة غير أمنة، نقوم بطباعة رسالة للمستخدم تخبره بأن الشبكة غير أمنة ونوقف التشغيل. وإذا كانت الشبكة مأمونة، نطبع رسالة تخبر المستخدم بأن الشبكة مأمونة ويمكن الاتصال بها. 

مثال آخر هو كتابة كود برمجي للبرامج الضارة:

```arabic
عيّن "البرنامج_الضار" كملف {
    إذا "الملف" مصاب {
        رسالة("الملف مصاب ببرنامج ضار، يجب حذفه")
        حذف "الملف"
    } وإلا {
        رسالة("الملف آمن")
    }
}
```

في هذا الكود، نبدأ بتعيين "البرنامج_الضار" كملف، ثم نتحقق إذا كان الملف مصاب ببرنامج ضار أم لا، و بناء على النتائج، نقوم بإصدار رسالة بحالة الملف والتعامل معه بناءً عليه.
