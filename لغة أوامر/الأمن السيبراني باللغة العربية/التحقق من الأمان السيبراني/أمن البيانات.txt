أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

### الأمر البرمجي: تحقق من الأمان السيبراني

في البرمجة باللغة العربية، يمكننا استخدام الإمرة `تحقق_من_الأمان(البيانات)` للقيام بفحص الأمان على بيانات معينة:

```arabic
دالة تحقق_من_الأمان(البيانات) {
    إذا (فحص_البيانات(البيانات) == خطأ) {
        أطبع("البيانات غير آمنة")
    } وإلا {
        أطبع("البيانات آمنة")
    }
}
```

هذا الأمر يقوم بعمل فحص على البيانات المدخلة وإعلام المستخدم إذا كانت هذه البيانات آمنة أو لا.

### مثال على تنفيذ الأمر البرمجي

```arabic
بياناتي = "معلومات حساسة"
تحقق_من_الأمان(بياناتي)
```

في هذا المثال، نحن نقوم بإدخال بيانات حساسة ونرغب في التحقق من أمان هذه البيانات. الناتج المتوقع من تنفيذ الأمر البرمجي سيكون إما "البيانات آمنة" أو "البيانات غير آمنة" حسب نتائج فحص الأمان.
