أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# التحقق من أمن البريد الإلكتروني

الكود البرمجي المهم مع تحقق أمن البريد الإلكتروني يجب أن يستطيع الإعتراض على الرسائل الغير أمنة والمشبوهة. في الأدنى يمكن كتابة:

```python
def تحقق_أمن_البريد(البريد_الإلكتروني):
    المرسل = البريد_الإلكتروني.أعط_المرسل()
    الموضوع = البريد_الإلكتروني.أعط_الموضوع()
    if المرسل in قائمة_المرسليين_المشبوهة or الموضوع in قائمة_المواضيع_المشبوهة:
        return 'مشبوه'
    else:
        return 'آمن'
```
و طريقة تطبيق هذا الكود البرمجي:
```python
قائمة_المرسليين_المشبوهة = ['مشبوه1@example.com', 'مشبوه2@example.com']
قائمة_المواضيع_المشبوهة = ['أنت الفائز', 'تم اختراق حسابك']
بريد_الإلكتروني = البريد('info@example.com', 'رسالة جديدة')
نتيجة = تحقق_أمن_البريد(البريد_الإلكتروني)
print(نتيجة)  # إذا كان البريد الإلكتروني آمن ، سوف يطبع 'آمن'
```

ملاحظة: هذا الكود البرمجي هو مثال بسيط وليس له القدرة على تأمين البريد الإلكتروني بشكل كامل. لهذه الغاية، يجب استخدام برامج متقدمة مخصصة للأمن السيبراني والبريد الإلكتروني.
