أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

كودمان (كود مان) هي اللغة البرمجية التي ابتكرناها لتكون باللغة العربية الفصحى! هذه أحد الجمل البرمجية في لغة كودمان:

```
واذا (كلمة_السر == "أمان") {
   اطبع("تم التحقق بنجاح")؛
} وإلا {
   اطبع("فشل التحقق")؛
}
```

في الكود البرمجي أعلاه، نقوم أولاً بالتحقق مما إذا كانت `كلمة_السر` تساوي "أمان". إذا كانت الاستجابة أجابة صحيحة، يتم طباعة الرسالة "تم التحقق بنجاح". في حالة عدم مطابقة `كلمة_السر` لـ"أمان"، يتم طباعة الرسالة "فشل التحقق".
