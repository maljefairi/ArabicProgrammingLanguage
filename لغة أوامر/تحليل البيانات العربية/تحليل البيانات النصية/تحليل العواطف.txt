أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بالحديث عن تحليل العواطف في البيانات النصية باللغة العربية، الفكرة الأساسية هي تقييم النص المدخل كيف يترجم شكلاً من أشكال الإحساس مثل الفرح، الحزن، الغضب، التحفيز والنشوة وغيرها. هنا مثال على كود بسيط لتحقيق هذه المهمة باستخدام البرمجة باللغة العربية:

```javascript
دالة تحليل_العواطف (نص) {
    نتائج = []
    قائمة العواطف = ["الفرح", "الحزن", "الغضب", "الخوف", "المفاجأة"]
    for (كل عاطفة في قائمة العواطف) {
        إذا ( النص يحتوي على عاطفة) {
            أضف إلى نتائج (عاطفة)
        }
    }
    إرجاع نتائج
}
```

الآن دعونا نطبق هذه الدالة مع نص بسيط:

```javascript
نص = "كان اليوم صعبا جدا، شعرت بالحزن والغضب"
تحليل_العواطف (نص)
// الناتج سيكون: ["الحزن", "الغضب"]
```

هذا مجرد مثال بسيط، وفي الحقيقية، الكود قد يكون أكثر تعقيدا ويحتاج لاستخدام الذكاء الصناعي والتعلم الآلي لتحليل العواطف بطريقة أكثر دقة وفعالية.
