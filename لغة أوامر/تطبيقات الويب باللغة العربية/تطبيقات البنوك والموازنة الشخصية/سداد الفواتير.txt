أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## سداد الفواتير: 

في بيئة برمجية باللغة العربية يمكن تعريف وظيفة لسداد الفواتير بهذه الطريقة:

```
دالة سداد_الفاتورة(رقم_الفاتورة, قيمة_السداد){
إذا (قيمة_السداد > رصيدي) {
  طباعة ("الرصيد غير كافي لسداد الفاتورة")
} وإلا {
    رصيدي = رصيدي - قيمة_السداد
    طباعة ("تم سداد الفاتورة بنجاح")
}
}
```

هذا الأمر البرمجي يستخدم لسداد قيمة الفاتورة من الرصيد الحالي. إذا كانت قيمة السداد أكبر من الرصيد الحالي، يتم طباعة رسالة تفيد بأن الرصيد غير كافٍ، وإلا، يتم طباعة رسالة تؤكد أن الفاتورة قد تم سدادها بنجاح. 

### لتطبيق هذا الأمر البرمجي:

مثلا لنفترض أن رقم الفاتورة هو 123 وقيمة السداد المطلوبة هي 200:

```
سداد_الفاتورة(123, 200)
```

النتيجة المتوقعة ستكون إما رسالة تفيد بأن الرصيد غير كاف أو رسالة تؤكد أن الفاتورة قد تم سدادها بنجاح، حسب قيمة الرصيد الحالي.
