أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

كود البحث عن رحلة بحرية:

```العربية
# هذا البرنامج يستخدم للبحث عن رحلة بحرية

دالة البحث_عن_رحلة_بحرية(المدينة_المغادرة, المدينة_الوصول, التاريخ) {
    رحلات = قاعدة_البيانات.احصل_على_الرحلات()
    رحلات_ممكنة = []

    لكل رحلة في رحلات {
        إذا رحلة.مدينة_المغادرة == المدينة_المغادرة و رحلة.المدينة_الوصول == المدينة_الوصول و رحلة.التاريخ == التاريخ {
            أضف رحلة إلى رحلات_ممكنة
        }
    }

    إرجاع رحلات_ممكنة
}
```

مثال على استخدام الكود:

```العربية
نتائج_البحث = البحث_عن_رحلة_بحرية("جدة", "القاهرة", "2021-12-01")
اطبع(نتائج_البحث)
```
في هذا المثال، نبحث عن رحلات بحرية مغادرة من جدة إلى القاهرة في التاريخ "2021-12-01". والنتائج تظهر عن طريق استدعاء الدالة `اطبع`.
