# البرمجة باللغة العربية الفصحى

الكود البرمجي في الغالب يكتب باللغة الإنجليزية لأنها مدعومة بشكل عالمي. ولكن من أجل تيسير العملية على المبرمجين العرب، لنفترض أننا قد أنشأنا لغة برمجة جديدة تعتمد على الجمل الصحيحة باللغة العربية. سأقترح ببساطة كيفية بناء جمل برمجية باللغة العربية.

### إنشاء متغير (variable):

لإنشاء متغير في لغتنا البرمجية الجديدة، سنستخدم الجملة "أنشئ متغير بأسم"

```
أنشئ متغير بأسم العمر وأعطه القيمة 25
```
تترجم هذه الجملة في اللغات البرمجية التقليدية إلى: `int age = 25;`

### إنشاء حلقة تكرار (loop):

لإنشاء حلقة تكرار، يمكننا استخدام الجملة  "كرر هذه العملية ... مرات"

```
كرر هذه العملية 10 مرات:
    - طباعة "مرحباً بك في لغة البرمجة العربية"
```
تترجم هذه الجملة في اللغات البرمجية التقليدية إلى:
```
for (int i=0; i<10; i++){
    System.out.println("مرحباً بك في لغة البرمجة العربية")
}
```

### الشرط (if statement):

```
إذا كان العمر أكبر من 18 :
    - طباعة "أنت قادر على التصويت"
وإلا
    - طباعة "أنت لست قادراً على التصويت"
```
تترجم هذه الجملة في اللغات البرمجية التقليدية إلى:
```
if (age > 18){
    System.out.println("أنت قادر على التصويت");
} else {
    System.out.println("أنت لست قادراً على التصويت");
}
```

حيث العمر هو المتغير الذي أنشأناه في البداية. 

يتم تصحيح هذه الجمل البرمجية باللغة العربية بنفس طريقة تصحيح الجمل البرمجية باللغة الإنجليزية في الأدوات المتوفرة مثل IDE ، حيث يُعرض تحذير إذا كان هناك خطأ في الجملة، ويمكن للمستخدم تصحيحه وفقًا لذلك.
