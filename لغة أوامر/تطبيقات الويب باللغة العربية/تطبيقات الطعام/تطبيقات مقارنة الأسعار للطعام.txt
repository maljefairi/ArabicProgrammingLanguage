أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

مراعاة أن هذه لغة برمجة مقترحة وليست لغة برمجة حقيقية أو تمت تنفيذها بالفعل. 

مثال على الحالة 1: تعريف المتغيرات
1. بالإنجليزية: `var price = 100;`
2. بالعربية: `متغير السعر = 100;`

مثال على الحالة 2: التحكم في التدفق
1. بالإنجليزية: 
``` 
if (price < 200) {
  alert('The price is affordable.');
} else {
  alert('The price is expensive.');
}
```
2. بالعربية: 
```
إذا (السعر < 200) {
  تنبيه('السعر معقول.');
} وإلا {
  تنبيه('السعر مرتفع.'); 
}
```

مثال على الحالة 3: الدوال
1. بالإنجليزية: `function calculateDiscount(price, discount) { return price - discount; }`
2. بالعربية: `دالة احسب_الخصم(السعر, الخصم) { ارجع السعر - الخصم; }`

مثال على الحالة 4: الحلقات
1. بالإنجليزية: 
```
for (var i = 0; i < 10; i++) { 
  console.log(i); 
}
```
2. بالعربية: 
```
لـ (متغير اي = 0; اي < 10; اي++) { 
  طباعة(اي); 
}
```

مثال على الحالة 5: المصفوفات
1. بالإنجليزية: `var fruits = ["apple", "banana", "mango"];`
2. بالعربية: `متغير الفواكه = ["تفاح", "موز", "مانجو"];`

وفي هذه الحالات، يمكن تطبيق هذه الأوامر البرمجية على سبيل المثال بناء تطبيق مقارنة الأسعار للطعام باستخدام الجمل البرمجية المقترحة باللغة العربية الفصحى. مثلاً، يمكن استخدام الدالة `احسب_الخصم` لحساب السعر الجديد للطعام بعد تطبيق الخصم. أو يمكنك استخدام الحلقة لعرض قائمة الأطعمة المتاحة في المصفوفة `الفواكه`.
