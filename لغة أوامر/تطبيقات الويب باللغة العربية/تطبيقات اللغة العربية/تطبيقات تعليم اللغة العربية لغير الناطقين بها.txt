أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بما أننا نتحدث عن انشاء لغة برمجة باللغة العربية الفصحى، سيكون من الجيد استخدام منهجية موجهة للكائنات. سأقترح كيف يمكن كتابة جملة برمجية على سبيل المثال لتعريف كائن باللغة العربية الفصحى.

لنفترض اننا نرغب في تعريف كائن جديد يُسمى "كتاب" وأن يحتوي هذا الكتاب على عنوان واسم المؤلف. في لغات البرمجة الشائعة مثل جافا أو C#، قد يكون الأمر مشابهًا لهذا:

```java
public class Book {
    private String title;
    private String author;

    //Constructor
    public Book(String title, String author) {
        this.title = title;
        this.author = author;
    }
    
    //Getters
    public String getTitle() {
        return title;
    }
    
    public String getAuthor() {
        return author;
    }
}
```

باللغة العربية، قد يكون الأمر مشابهًا لهذا:

```
# ها هو كود البرمجة باللغة العربية الفصحى لتعريف كائن جديد يُسمى "كتاب".

صنف كتاب {
    خاص عنوان نص;
    خاص مؤلف نص;

    # البناء
    بناء(عنوان نص, مؤلف نص) {
        هذا.عنوان = عنوان;
        هذا.مؤلف = مؤلف;
    }

    # الواجهات
    جلب عنوان() {
        رجع عنوان;
    }

    جلب مؤلف() {
        رجع مؤلف;
    }
}
```
أتمنى أن يكون هذا قد أعطى فكرة عن كيف يمكن أن تبدو لغة برمجة باللغة العربية. حاولت أن أجعل الكود قريبًا قدر الإمكان من الكود الأصلي بجافا مع الأخذ بالاعتبار بعض التغييرات التي قد تكون ضرورية للغة الجديدة. 

ملاحظة: إن الكود البرمجة العربي المشار اليه أعلاه ليس حقيقي و لا يمكن تنفيذه بعد لأننا لا نملك حقيقتا لغة برمجة باللغة العربية أثناء كتابة هذه الفقرة. ولكنه فقط توضيح كيف قد تبدو لغة البرمجة العربية بالنسبة لك.
