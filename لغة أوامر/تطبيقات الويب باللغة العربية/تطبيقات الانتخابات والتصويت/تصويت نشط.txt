أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# تصويت نشط

يتضمن البرمجة في العربية تقديم الكود بطريقة تتيح للمستخدمين الناطقين باللغة العربية فهمه بسهولة. لذلك، سأقوم بإنشاء كود يمثل "تصويت نشط" باستخدام لغة برمجة عربية.

فلنفترض أن لدينا موقع تصويت، حيث يتوجب علينا إنشاء كود يقوم بتمكين عملية التصويت وتمييز الأصوات المنشطة.

```ARABIC

# تعريف الكائن
نوع الكائن صوت {
    متغير نشط: بولين؛
    متغير معرف -المصوت: رقم؛
}

# تعريف الدالة
دالة تصويت_نشط ( ص: صوت) رجوع بولين {
    إذا (ص.نشط == صح) {
        رجوع صح؛
    } وإلا {
        رجوع خطأ؛
    }
}
```

في الكود أعلاه، أنشأنا كائن "صوت" يمتلك خاصيتين: "نشط" و "معرف_المصوت". ثم، أنشأنا دالة "تصويت_نشط" تقوم بفحص حالة الصوت إذا كان نشط أم لا.

أمثلة الاستعمال:

```ARABIC

# إنشاء كائن من نوع صوت
متغير ص = صوت {نشط: صح، معرف_المصوت: 1}؛

# استعمال الدالة
طباعة(تصويت_نشط(ص))؛ # هذا سيطبع: صح
```

هذا الكود المقترح مثال على كيفية استعمال "تصويت نشط" باللغة العربية.
