أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# انشاء لغة برمجة باللغة العربية الفصحى

البرمجة باللغة العربية الفصحى يمكن أن تكون تجربة فريدة وتمكن العرب من التعبير عن أفكار البرمجة بطريقة مألوفة لهم. نعتبر "أمر" كمثال لكيفية بناء كود برمجي باللغة العربية الفصحى.

```arabic
أمر طباعة = "مرحبا بك في البرمجة باللغة العربية الفصحى!"
طباعة(طباعة)
```

هذا الكود السابق يقوم بتعيين النص "مرحبا بك في البرمجة باللغة العربية الفصحى!" إلى المتغير "طباعة"، ثم يقوم بطباعة النص عن طريق استدعاء الدالة "طباعة".

# البرمجة الوظيفية باللغة العربية

البرمجة الوظيفية هي نهج يركز على استخدام الدوال. في اللغة العربية الفصحى، يمكن أن تبدو هكذا:

```arabic
دالة جمع(أ, ب){
  أ + ب
}

أمر المجموع = جمع(5, 3)
طباعة(المجموع)
```
هذه الشيفرة تعرف دالة تُسمى "جمع" تأخذ مُدخلين، يُسميان "أ" و"ب". هذه الدالة تجمع القيمتين وتعيدها. ثم يتم استدعاء الدالة لجمع 5 و3 وتخزين القيمة الناتجة في "المجموع" وأخيراً يتم طباعة "المجموع". 

# البرمجة باللوغاريتمات في اللغة العربية

بالإمكان التعامل مع اللوغاريتمات في اللغة العربية الفصحى بنفس الطريقة التي نتعامل بها مع الأرقام والدوال الأخرى، مثلا:

```arabic
دالة اللوغاريتم(قاعدة, قيمة) {
  إذا قاعدة > 0 و قاعدة != 1 و قيمة > 0 {
    نتيجة = لوغاريتم(قاعدة, قيمة)
    طباعة(نتيجة)
  } وإلا {
    طباعة("المدخلات غير صحيحة")
  }
}

اللوغاريتم(2, 8)
```
هذا الكود يعرف دالة اللوغاريتم التي تستقبل قاعدة وقيمة لحساب اللوغاريتم، ويتم التحقق من أن القاعدة والقيمة هما مدخلات صحيحة، ثم يتم حساب اللوغاريتم وطباعة النتيجة. في النهاية، تم استدعاء الدالة مع قاعدة 2 وقيمة 8.
