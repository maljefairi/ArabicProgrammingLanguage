أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بناء على ما فهمت من السياق، نحن نقوم بتطوير لغة برمجة باللغة العربية. البرمجة الوظيفية والبرمجة باللوغاريتمات والتحسين القائم على البحث هي المميزات الرئيسية لهذه اللغة.

مثال على الكود البرمجي باللغة العربية الفصحى لإيجاد العدد الأولي المتتابع:

```عربي
دالة هل_أولي(العدد) {
    إذا العدد <= 1 أعد خطأ;
    إذا العدد <= 3 أعد صواب;
    
    كرر(ك كما عدد من 2 إلى جذر_تربيعي(العدد)) {
        إذا العدد % ك == 0 أعد خطأ;
    }
    
    أعد صواب;
}

دالة الأوليين_المتواليين(العدد) {
    حدد قائمة أعداد أولية = [];

    كرر(ك كما عدد من 2 حتى حجم(أعداد_أولية) < العدد) {
        إذا هل_أولي(ك) أضف ك إلى أعداد_أولية;
    }
    
    أعد أعداد_أولية;
}
```
تطبيق هذا الكود على مشروع حيث يحتاج المطور فيه لإيجاد الأعداد الأولية المتوالية سيكون مثل الأتي: نستدعي الدالة `الأوليين_المتواليين` كالتالي:
```عربي
أطبع الأوليين_المتواليين(5);
```
و هذا الأمر في البرنامج البرمجي سيطبع الأعداد الأولية الخمسة الأولى.
