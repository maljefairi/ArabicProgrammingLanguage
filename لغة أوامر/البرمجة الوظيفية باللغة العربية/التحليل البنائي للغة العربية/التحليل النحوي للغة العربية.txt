أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

انشاء لغة برمجة باللغة العربية الفصحى يتطلب فهماً عميقاً لهيكل الجمل العربية وقواعدها. لنفترض أننا نرغب في انشاء لغة برمجة بسيطة تعمل على تطبيق مفهوم "الدوران" أو "التكرار".  نستطيع تصميم لغة برمجة تستخدم العبارة "أعد_التكرار" وتستخدم الأرقام العربية للتحكم في عدد المرات التي يتم فيها التكرار. لاحقا~, يمكن ان يتم قراءة جملة برمجية مثل "أعد_التكرار 10 مرات". 

هذا هو شكل الكود البرمجي المقترح:

```العربية
الدالة اطبع (رسالة) {
    اكتب في الشاشة (رسالة);
}

الدالة أعد_التكرار ( عدد_المرات , الأمر ) {
    لـ (جداول = 0; جداول < عدد_المرات ; جداول ++) {
        استدعي الأمر;
    }
}

البرنامج الرئيسي {
    اكتب في الشاشة("البدء بالبرنامج...");
    أعد_التكرار (10, () => اطبع ("تجربة") );
}
```

في الكود أعلاه, الدالة `أعد_التكرار` تقبل قيمة رقمية وأمراً لتنفيذه. القيمة الرقمية تحدد عدد مرات تنفيذ الأمر، والأمر المراد تنفيذه هو دالة تطبع الكلمة "تجربة" في كل تكرار.
