أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# لغة البرمجة العربية

نقدم لكم هنا نموذج لكود برمجي باللغة العربية الفصحى يتضمن مثال لكيفية تطبيق هذا الأمر بناءً على لغة البايثون:

إنشاء الكائنات:
```arabic
كائن سيارة:
   الخاصية اللون
   الخاصية النوع
   دالة تشغيل:
      اكتب("السيارة تعمل الآن")
سيارة نيسان = سيارة("أزرق", "سيدان")
نيسان.تشغيل()
```

لتعريف الدوال:
```arabic
دالة جمع(أ, ب):
   رجوع أ + ب
جمع(7, 3)
```

للتكرار و الفروع:
```arabic
لكل رقم في نطاق(1, 11):
   إذا الرقم % 2 == 0:
      اكتب(رقم, "هو زوجي")
   وإلا:
      اكتب(رقم, "هو فردي")
```

للقوائم و القواميس:
```arabic
قائمة العناصر = [1, 2, 4, 7]
قاموس الفواكه = { "تفاح": 10, "موز": 5, "برتقال": 3 }
For فاكهة, العدد في الفواكه.أصناف():
    اكتب("يوجد", العدد, "من", الفاكهة)
```

لهذا الكود، يتم تطبيق الأوامر بشكل مباشر وهو يستخدم أوامر غالباً ما تكون ثابتة مثل كائن، الخاصية، دالة، اكتب، رجوع، إذا، وإلا. بينما القيم التي يجب أن تكون متغيرة مثل "سيارة"، "أزرق"، "تشغيل"، "جمع"، "أ"، "ب"، يمكن تغييرها حسب الحاجة في الكود البرمجي.
