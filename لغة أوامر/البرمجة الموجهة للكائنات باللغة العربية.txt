# البرمجة الموجهة للكائنات باللغة العربية

## 1. التعريف بكلاس جديد
```arabic
كلاس عامل {
    حقل اسم;
    حقل راتب;
}
```
الكود أعلاه يعرف كلاس جديد يدعى "عامل" بداخله حقلين، الأول "اسم" والثاني "راتب".

## 2. إنشاء كائن جديد
```arabic
عامل عامل1 = جديد عامل();
عامل1.اسم = "محمد";
عامل1.راتب = 5000;
```
هنا نقوم بإنشاء كائن جديد من الكلاس "عامل" وتعيين قيم لحقلي "اسم" و"راتب".

## 3. تعريف وتنفيذ الدوال
```arabic
كلاس عامل {
    حقل اسم;
    حقل راتب;
    
    دالة رفع_الراتب(قيمة) {
        راتب = راتب + قيمة;
    }

    دالة الراتب() {
        رجوع راتب;
    }
}

عامل عامل1 = جديد عامل();
عامل1.اسم = "محمد";
عامل1.راتب = 5000;
عامل1.رفع_الراتب(500);
طباعة(عامل1.الراتب()); // بالهندسة القديمة سيطبع «5500»
```
نعريف دالة جديدة "رفع_الراتب" تقوم بزيادة الراتب بمقدار ما تم تحديده بداخل "قيمة". وأيضا نعرف دالة تسمى "الراتب" تقوم بإرجاع الراتب للعامل. في الأخير نقوم بتطبيق هاتين الدالتين على العامل1.
