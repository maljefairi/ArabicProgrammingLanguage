أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إنشاء لغة برمجة باللغة العربية تقوم بتصحيح الأخطاء وتستخدم الجمل الصحيحة ليس بالمهمة البسيطة. ولكن دعونا نتخيل كيف قد يكون شكل بعض الأوامر الأساسية في لغة البرمجة هذه:

1. **طباعة** - لعرض النصوص أو الأرقام على الشاشة:
```arabic
طباعة("مرحبا بكم في برمجة باللغة العربية");
```

2. التحقق من الشرط - لفحص ما إذا كانت الشروط معينة صحيحة أم لا:
```arabic
إذا (عمر > 18) {
    طباعة("أنت قادر على التصويت");
} وإلا {
    طباعة("أنت غير قادر على التصويت");
}
```
3. **لكل** - لتنفيذ كود عدة مرات وفقًا للشرط المعين:
```arabic
لكل (عدد في الأرقام) {
    طباعة(عدد);
}
```
4. **تعريف** - لتعريف دالة جديدة تقوم بفحص ما إذا كان الرقم أولي أم لا:
```arabic
تعريف هل_أولي(رقم) {
    إذا (الرقم <= 1) {
       طباعة("ليس رقمًا أوليًا");
       إرجاع;
    }
    لكل (عدد = 2; عدد < الرقم; عدد++) {
       إذا (الرقم % الرقم == 0) {
          طباعة("ليس رقمًا أوليًا");
          إرجاع;
       }
    }
    طباعة("هو رقم أولي");
}
```

هذه فقط بعض الأمثلة على كيفية استخدام الكلمات العربية في كتابة البرمجة بينما تبقى الجمل ذات معنى واضح للمتحدثين باللغة العربية. المهم هو أن يتم التأكد من أن القواعد اللغوية محترمة والجمل مفهومة وبسيطة بقدر الإمكان.
