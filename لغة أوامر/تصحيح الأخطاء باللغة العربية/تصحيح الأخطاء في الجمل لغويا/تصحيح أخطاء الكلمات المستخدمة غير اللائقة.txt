أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إذا كانت لغتك البرمجية تدعى "أبجدية"، فقد يكون الكود البرمجي الخاص بك مشابهًا لهذا:

```أبجدية
إذا (المتغير == القيمة) {
    أكتب( "الجملة صحيحة" );
} وإلا {
    أكتب( "الجملة غير صحيحة" );
}
```
يشير الجزء "إذا (المتغير == القيمة)" إلى شرط معين يجب تحققه. وفي هذا السياق، إذا كان "المتغير" مساويًا ل "القيمة"، يتم تنفيذ جملة البرنامج النصية داخل الأقواس {} التالية، والتي تقول "أكتب( "الجملة صحيحة" )".

وإلا، إذا كان الشرط غير صحيح، يتم تنفيذ الجملة "أكتب( "الجملة غير صحيحة" )".

لإعطاء مثال على كيفية تطبيق هذا الأمر: إذا كنا نرغب في التحقق مما إذا كان العدد في المتغير "العدد" يساوي 10، يمكننا القيام بما يلي:

```أبجدية
إذا (العدد == 10) {
    أكتب( "العدد هو 10" );
} وإلا {
    أكتب( "العدد ليس 10" );
}
```
في هذه الحالة، ستقوم اللغة البرمجية بكتابة "العدد هو 10" إذا كان المتغير "العدد" يحتوي على القيمة 10. وإلا، ستقوم بكتابة "العدد ليس 10".
