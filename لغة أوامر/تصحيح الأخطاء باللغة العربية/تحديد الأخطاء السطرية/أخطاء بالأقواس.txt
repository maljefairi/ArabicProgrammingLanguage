أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## إدخال قيمة

```
أدخل (متغير)
```
هذا الأمر يتيح الفرصة للمستخدم لإدخال قيمة.

**مثال**: 

إذا كان لديك متغير يدعى "عمر" وتريد تحديد قيمته بناء على ما يدخله المستخدم، يمكنك كتابة كود مثل هذا:

```
أدخل (عمر)
```

## عرض القيمة 

```
عرض (متغير)
```
هذا الأمر يقوم بعرض قيمة المتغير.

**مثال**:

 إذا كان لديك متغير يدعى "اسم" وتريد عرضه، يمكنك كتابة كود مثل هذا:

```
عرض (اسم)
```

## التحقق من المدخلات

```
إذا (شرط) – فعل – و إلا 
```
هذا الأمر يتيح القدرة على فحص الشروط.

**مثال**:

إذا كنت تريد التحقق من ما إذا كان عمر المستخدم يزيد عن 18 سنة، يمكنك كتابة كود مثل هذا:

```
إذا (عمر > 18) 
– عرض (أنت مسموح لك)
و إلا 
– عرض (أنت غير مسموح لك)
``` 

## تحديد الخطأ بالأقواس 

```
أخطاء بالأقواس
``` 
وظيفة هذا الأمر تكمن في تقديم إشارة إلى المستخدم في حالة وجود أخطاء في الأقواس. على سبيل المثال، إذا تم إغلاق الأقواس بشكل غير صالح أو تم تجاوز عدد الأقواس المفتوحة عن عدد الأقواس المغلقة.

**مثال**:

إذا كان لديك كود مثل هذا:

```
إذا (عمر > 18
```
فسيظهر هذا الخطأ: ```أخطاء بالأقواس```، لأن الأقواس غير مكتملة.

أما في حالة إكمال الأقواس:

```
إذا (عمر > 18)
```
فسيتم تنفيذ الكود بصورة سليمة.
