أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

أكتب البرمجة المقترحة باللغة العربية الفصحى: 

نظراً لطبيعة هذه السياق التي تركز على تطبيقات حجز الفنادق، يمكن استخدام الكود التالي كمثال. 

يمكن تقديم لغة برمجة باللغة العربية على النحو التالي:

```
أنشئ واجهة اسمها "واجهة حجز الفنادق" {
  حقل مدخلات "اسم الفندق"
  حقل مدخلات "تاريخ الوصول"
  حقل مدخلات "تاريخ المغادرة"
  زر اسمه "حجز" {
    عند الضغط على "حجز" {
      اطبع "تم حجز الفندق"
    }
  }
}
```
البرنامج البسيط المذكور أعلاه ينشئ واجهة بسيطة تتكون من ثلاث حقول لإدخال معلومات الفندق، بالإضافة إلى زر "حجز". عند الضغط على الزر، يظهر رسالة تؤكد اكتمال عملية الحجز.
