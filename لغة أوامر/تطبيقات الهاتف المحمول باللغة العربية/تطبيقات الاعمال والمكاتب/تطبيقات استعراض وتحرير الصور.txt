# تعريف اللغة البرمجية باللغة العربية الفصحى
نسعى ايضا لتقديم أساسيات اللغة البرمجية باللغة العربية الفصحى، بحيث يمكن للمستخدم التعامل مع المواقف البرمجية بشكل صحيح وفصيح.

### عملية الطباعة الأساسية
لنبدأ بمثال بسيط على طباعة النصوص. فبدلا من استخدام امر "print" الشائع في اللغات البرمجية، سنستخدم الكلمة العربية "اطبع" كما يلي:

```
اطبع "مرحبا بك في لغة البرمجة العربية"
```
هذا يعادل الأمر البرمجي بالإنجليزية:
``` 
print("Hello in Arabic Programming Language") 
```

### العمليات الرياضية
بالنسبة للعمليات الرياضية، نستخدم الكلمات العربية للتعبير عن العمليات، مثل "اجمع" و"اطرح" و"اضرب" و"اقسم". ها هو مثال:

```
عددأ = ١٠
عددب = ٢٠
اجمع (عددأ, عددب)
```

هذا يعادل الأمر البرمجي بالإنجليزية:
``` 
numberA = 10
numberB = 20
sum(numberA, numberB)
```

### وصولا الى تطوير تطبيقات استعراض وتحرير الصور 
بإمكاننا أيضا تطوير تطبيقات استعراض وتحرير الصور باللغة العربية الصحيحة و ايضا ذكر مثال على ذلك :

```
صورة = افتح("الطريق.jpg")  
معالجة(صورة)
```

هذا يعادل الأمر البرمجي بالإنجليزية:
``` 
image = open("path.jpg")
process(image)
```

الفكرة الأساسية هنا هي استخدام اللغة العربية الصحيحة والفصحى في تطبيق العمليات البرمجية بدلا من اللغة الإنجليزية المعتادة أو أي لغة برمجية أخرى. 

فكرة هذه اللغة البرمجية العربية هي اعطاء القدرة للمتكلمين باللغة العربية لفهم البرمجة وتعلمها بشكل اسهل و اكثر فهما.
