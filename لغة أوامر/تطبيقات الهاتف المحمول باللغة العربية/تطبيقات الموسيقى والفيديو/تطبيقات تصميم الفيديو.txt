# تصميم الفيديو باللغة العربية
إذا كنا نرغب في إنشاء تطبيق لتصميم الفيديو، نستطيع أن نستخدم واحدة من لغات البرمجة الشائعة مثل الجافا أو السويفت، ولكن بما أننا نرغب في استخدام اللغة العربية الفصحى، سنبدأ بتحويل بعض الجمل البرمجية الشائعة في تصميم الفيديو إلى اللغة العربية.

للبدء، سنضع بعض الأمثلة البسيطة.  مثلا, إذا أردنا تنفيذ التعليمة "إذا" في لغة البرمجة، سيكون كود الجافا كما يلي:

```java
if (condition) {
    // statements to be executed if the condition is true
}
```

وفي اللغة العربية الفصحى، هذا قد يمكن ترجمته إلى:

```arabic
إذا (شرط) {
    // الأوامر التي يجب تنفيذها إذا كان الشرط صحيح
}
```

إذا أردنا القيام بعملية تكرار في البرمجة والتي تعبر عنها الجملة "لكل" في اللغة العربية، في الجافا سيكون كما يلي:

```java
for (int i = 0; i < n; i++) {
    // statements to be executed for each iteration
}
```

وفي اللغة العربية الفصحى، هذا قد يمكن ترجمته إلى:

```arabic
لكل (عدد صحيح ا = 0; ا < ن; ا++) {
    // الأوامر التي يجب تنفيذها في كل تكرار
}
```

وبالتالي، يمكننا تطبيق هذه التعليمات البرمجية في تصميم تطبيق الفيديو باللغة العربية الفصحى.
