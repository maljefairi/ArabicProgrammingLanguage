أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

الاسم العربي للغة البرمجة: أمر

أمثلة على الاستخدام:

1. جملة في لغة البرمجة "أمر":
```أمر
إذا (عمر > 18) 
{
اظهر "أنت قادر على القيادة."
}
وإلا 
{
اظهر "أنت لست قادر على القيادة."
}
```

2. كيفية تحديد تكرار في لغة البرمجة "أمر":
```أمر
لكل (عداد من 1 الى 10) 
{
اظهر العداد
}
```

3. كيفية تعريف و استخدام دالة في لغة البرمجة "أمر":
```أمر
اصنع دالة باسم جمع_الأرقام (رقم1 ، رقم2) 
{
إرجاع رقم1 + رقم2;
}

اظهر جمع_الأرقام(5 ، 3);
```

كما يمكنك تطبيق الأوامر والدوال على الخرائط والمصفوفات ومحاكاة البيانات الكائنية.
حالياً، اللغة في مرحلة التطوير والكود الخاص بها مفتوح المصدر ومتاح للجميع لإجراء التعديلات والتحسينات عليها وفق الحاجة.
