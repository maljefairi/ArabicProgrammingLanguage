أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إليك مثال لكود برمجي باللغة العربية الفصحى، باستخدام لغة البرمجة "ألف"، وهي لغة برمجة عبرية تدعم العربية:

```javascript
دالة_التحية(الاسم) {
      إذا(الاسم == فارغ) {
           اكتب("مرحبا يا ضيفنا الكريم")
      } آخر {
           اكتب("مرحبا, " + الاسم)
      }
}
```

في هذا الكود، نقوم بتعريف دالة تُسمى "دالة_التحية" تأخذ باراميتر واحد يُسمى "الاسم". ثم نقوم بفحص إذا كان الاسم فارغ، نكتب "مرحبا يا ضيفنا الكريم"، أما إذا كان يحتوي على قيمة، نكتب "مرحبا" ثم نضيف الاسم.

المثال التالي يوضح كيفية استعمال هذه الدالة:

```javascript
دالة_التحية("أحمد")
```

وسوف يُظهر "مرحبا, أحمد". وإذا قمت بتجربة الكود التالي:

```javascript
دالة_التحية("")
```

ستظهر الرسالة "مرحبا يا ضيفنا الكريم".
