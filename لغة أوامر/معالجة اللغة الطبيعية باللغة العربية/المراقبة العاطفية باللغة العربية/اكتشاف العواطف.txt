أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

تصميم لغة برمجة باللغة العربية له أوجه تحدي عديدة نظرًا لتعقيد اللغة والفروق الدقيقة في الجمل والعبارات. ولكن، بدلاً من تصميم لغة برمجة بالكامل، فإنه يمكن تطبيق جزء من الجمل البرمجية في لغة البرمجة الموجودة مثل Python باللغة العربية. 

على سبيل المثال، اعتبر أنك تود إنشاء دالة بالعربية في Python مما يمكن استخدامه كجزء من لغة البرمجة العربية المقترحة. 

يمكن لكتابة الأمر البرمجي لاكتشاف العواطف ببيانات النص باللغة العربية أن يبدو هكذا:

```python
# تعريف الدالة
def اكتشاف_العواطف(نص: str) -> str:
    عواطف = {'الفرح': ["سعيد", "فرحان"], 'الغضب': ["غاضب", "معصب"]}
    لكل عاطفة, الكلمات في عواطف.items():
        لكل كلمة في الكلمات:
            إذا كلمة في نص:
                ترجع عاطفة
    ترجع "عاطفة غير معروفة"

# استخدام الدالة
نتيجة = اكتشاف_العواطف("أنا سعيد اليوم")
print(نتيجة)  # يطبع: الفرح
```

لكن يجب الانتباه إلى أن الكود البرمجي المقترح لن يعمل فعليًا في Python كما هو الآن لأن Python لا يدعم الكلمات الأساسية باللغة العربية. ولكن يمكن اعتباره فقط مثالاً توضيحياً على كيفية البرمجة باللغة العربية.
