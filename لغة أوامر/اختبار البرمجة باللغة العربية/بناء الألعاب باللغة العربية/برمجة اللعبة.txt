# برمجة لعبة باللغة العربية

في هذا السياق، سنحاول إنشاء لعبة بسيطة في أن تتحرك الشخصية من اليمين إلى اليسار باللغة العربية الفصحى. سنستخدم لغة برمجة تم تطويرها خصيصًا للكتابة باللغة العربية. نتصور القواعد التالية للبرمجة باللغة العربية: 

1. `ابدأ ب` : يعبر عن بداية تنفيذ برنامج أو نظام برمجي.
2. `وظيفة` : تعتبر البنية الأساسية لتعريف الوظائف / الطرق.
3. `قل ل` : تعبر عن طباعة أو عرض شيء على الشاشة.
4. `انتقل إلى` : تعبر عن تحريك العنصر أو الشخصية في اللعبة.

الكود البرمجي باللغة العربية سيبدو تقريبًا بالشكل التالي:

```
ابدأ ب
  وظيفة بداية_اللعبة {
    قل ل "مرحبًا بك في اللعبة! الرجاء تحريك الشخصية بإتجاه اليسار"
    انتقل_إلى الشخصية في الخلفية(الموقع(0,0))
  }

  وظيفة انتقل_إلى(العنصر, الموقع) {
    العنصر.انتقل_إلى الموقع
  }
نهاية ب
```

في هذا المثال نستخدم وظيفتين. الوظيفة الأولى `بداية_اللعبة` تظهر رسالة الترحيب باللاعب وتحريك الشخصية إلى الموقع الأول. الوظيفة الثانية `انتقل_إلى` هي الوظيفة التي تسمح لنا بتحريك العنصر (في هذه الحالة الشخصية) إلى الموقع المطلوب.
