أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إحدى الأفكار التي يمكن استخدامها في بناء لغة البرمجة بالعربية الفصحى هو استعمال الأوامر بشكل خاضع للقواعد النحوية في العربية. فمثلاً، بدل من كتابة أمر مثل:

```
if ( س = 5 ) {
println("القيمة صحيحة");
} 
```

نمشي بأسلوب أكثر أدبية وفصاحة في العربية، كأن نقوم بتعريف الأمر كذلك:

```
إن كانت القيمة س تساوي ٥، فاكتب "القيمة صحيحة".
```

أو حتى في الإجراءات التكرارية، بدلًا من:

``` 
for (i = 0; i < 10; i++) {
println(i);
}
```

يُصبح هكذا:

```
طبعا لآرقام إعادة من صفر حتى تسعة جد الناتج. 
```

حيث أصبح الأمر "جد الناتج" هو الأمر لطباعة القيمة. وعلى النحو التالي نستطيع أن نطور من تلك الفكرة ونصنع لغة برمجة باللغة العربية الفصحى تستخدم الجمل الصحيحة.
