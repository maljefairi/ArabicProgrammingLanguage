أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## مستندات اللغة البرمجة باللغة العربية

### تحديد متطلبات المشروع 

هذا المفهوم يتعلق بالعمل المطلوب لتقدير حجم المشروع والميزات التي يحتاجها. لتنفيذ هذه العملية باللغة البرمجة العربية، نستخدم الجملة التالية :

عين `متطلبات_المشروع`

مثال :

{% code language="Arabic" %}
عين متطلبات_المشروع
{
  "واجهة المستخدم": {"نوع الواجهة": "طبيعية"},
  "أمان البيانات": {"مستوى الأمان": "عالي"},
  "اداء البرنامج": {"السرعة": "سريع"},
  "التحقق من الأخطاء": {"تصحيح_أوتوماتيكي": "نعم"}
}
{% endcode %}

### تطبيق القواعد والمعايير البرمجية

هنا نحتاج إلى تطبيق معايير البرمجة المتفق عليها في المشروع. في اللغة البرمجة العربية، يتم تطبيق القواعد والمعايير بواسطة الجملة التالية:

طبق `القواعد والمعايير`

مثال :

{% code language="Arabic" %}
طبق القواعد والمعايير
{
  "التعليقات": {"تواجد التعليقات في الكود": "نعم"},
  "تسمية المتغيرات والدوال": {"استخدام الأسماء المعبرة": "نعم"},
  "هيكل الكود": {"التنظيم المناسب": "نعم"},
  "الأداء": {"التحقق من أداء الكود": "نعم"},
  "اختبار الكود": {"اجراء اختبارات شاملة للكود": "نعم"},
  "الأمان": {"التحقق من أمان الكود": "نعم"}
}
{% endcode %}
إذا كان هناك أي أخطاء أو مخالفات للمعايير، ستظهر رسالة تنبيه.
