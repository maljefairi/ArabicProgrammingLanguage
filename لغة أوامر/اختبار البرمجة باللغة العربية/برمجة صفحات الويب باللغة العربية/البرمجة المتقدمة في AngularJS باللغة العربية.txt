أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## البرمجة المتقدمة في AngularJS باللغة العربية

### باللغة العربية الفصحى، سنقوم بتكوين تطبيق AngularJS بسيط. دعونا نؤسس النموذج "طلب" في الكود التالي:

```جافا سكريبت
التطبيق.التحكم("طلب", دالة($المجال) {
  $المجال.الأصناف = [{
      اسم: "التفاح",
      الكمية: 10
    },
    {
      اسم: "البرتقال",
      الكمية: 20
    },
    {
      اسم: "الموز",
      الكمية: 30
    }
  ];
});
```

### ومن ثم، سنقوم بتقديم البيانات المحددة في واجهة المستخدم مع الكود التالي:

```جافا سكريبت
<div ng-app="تطبيق" ng-controller="طلب">
  <ul>
    <li ng-repeat="صنف in الأصناف">
      {{ صنف.اسم }}: {{ صنف.الكمية }}
    </li>
  </ul>
</div>
```

### في الكود الاخير، تستخدم واصفة `ng-repeat` لاستعراض كل "صنف" في "الأصناف"، ولطباعة "الاسم" و "الكمية" لكل "صنف".
### عند تشغيل هذا الكود، سيقوم بعرض قائمة بالأصناف والكميات المتوفرة.
