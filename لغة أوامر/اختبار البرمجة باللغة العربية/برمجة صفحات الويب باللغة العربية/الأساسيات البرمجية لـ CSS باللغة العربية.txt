# الأساسيات البرمجية لـ CSS باللغة العربية

لنفترض أننا نريد إضافة لون خلفية للصفحة وتحديد اللون الخاص بالنص في موقع الويب الخاص بنا. نستطيع أن نعبر عن هذا الأمر في CSS باللغة العربية على النحو التالي:

```css
صفحة {
 غلفية_لون: أبيض;
 لون: أسود;
}
```
الجملة البرمجية أعلاه هي نفس الجملة البرمجية في النموذج الإنجليزي الأصلي لـ CSS والتي تقول:

```css
body {
 background-color: white;
 color: black;
}
```
كلا من الأمرين يقومون بنفس الوظيفة، ولكن الأمر الأول مكتوب باللغة العربية الفصحى وهو أيضا مفهوم بوضوح للقارئ الناطق بالعربية.

يتم تحويل الكود البرمجي المكتوب باللغة العربية إلى كود برمجي قابل للتنفيذ بواسطة مترجم خاص يمكنه فهم الأوامر المكتوبة باللغة العربية وترجمتها إلى الأوامر المقبولة في لغة برمجة مثل CSS. لذا، يمكن لأي شخص يتحدث اللغة العربية أن يتعلم كتابة الأوامر البرمجية باستخدام اللغة العربية الفصحى.
