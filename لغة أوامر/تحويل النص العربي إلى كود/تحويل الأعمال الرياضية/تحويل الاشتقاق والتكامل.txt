أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## تحويل الأوامر البسيطة
يمكن بسهولة تحويل أوامر برمجة بسيطة باللغة الإنجليزية إلى العربية الفصحى. 

إليك مثال بسيط: 

**جافا**
```java
System.out.println("مرحباً بك في اللغة العربية!");
```
**تحول إلى**
```java
أطبع("مرحباً بك في اللغة العربية!");
```

## تحويل الجمل الشرطية
الجمل الشرطية واحدة من أساسيات البرمجة. من المهم أن نحولها إلى العربية بطريقة تسهل فهمها.

**جافا**
```java
if (x > 1) {
    System.out.println("X أكبر من 1");
}
```
**تحول إلى**
```java
إذا (س > 1) {
    أطبع("س أكبر من 1");
}
```

## تحويل الأعمال الرياضية
من الضروري أن يكون الكود البرمجي العربي قادراً على لفهم وأداء الأعمال الرياضية 

**جافا**
```java
int y = (a + b) * (c / d);
```
**تحول إلى**
```java
عدد ص = (أ + ب) * (ج / د);
```

## تحويل الاشتقاق والتكامل
لأن البرمجة تستخدم بشكل واسع في مجالات المعرفة كالرياضيات، لذلك يجب تحويل المبادئ الرياضية الأساسية أيضًا 

**جافا**
```java
double derivative = (f(x + h) - f(x)) / h;
double integral = (f(x) + f(x + h)) / 2 * h;
```
**تحول إلى**
```java
عدد مع الفاصلة المشتق = (أف(س + ه) - أف(س)) / ه;
عدد مع الفاصلة التكامل = (أف(س) + أف(س + ه)) / 2 * ه;
```
