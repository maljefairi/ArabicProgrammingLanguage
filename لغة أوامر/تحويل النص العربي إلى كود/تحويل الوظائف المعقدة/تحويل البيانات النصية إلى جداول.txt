أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# لغة برمجة بالعربية الفصحى 

يمكن في لغة البرمجة العربية التي سنطورها إنشاء متغيرات، وتمثيل الجمل الشرطية، وأيضا إنشاء الدوال والكائنات. 

**دعونا نعطي بعض الأمثلة:**

## إنشاء متغير
إعتبار أننا نريد تعريف متغير باسم "رقم". سنكتب بالعربية الفصحى الجملة التالية: 

```
اجعل الرقم هو 5
```
وسيتم تحويلها إلى كود البرمجة بالصورة التالية:
```javascript
let رقم = 5;
```

## الجمل الشرطية
نريد تطبيق جملة شرطية بحيث إذا كان الرقم أكبر من 10، نكتب جملة "الرقم كبير". 

بالعربية الفصحى، سنكتب: 

```
إذا كان الرقم أكبر من 10، قل "الرقم كبير"
```
والكود البرمجي المقابل سيكون:
```javascript
if(رقم > 10){
    console.log("الرقم كبير");
}
```

## إنشاء دالة
مثال على دالة تقوم بإرجاع نص. ستكتب:

```
اصنع دالة باسم "ترحيب" تقوم بقول: "مرحبا بك في لغة البرمجة العربية" 
```

وسيتم تحويلها إلى:
```javascript
function ترحيب(){
    return "مرحبا بك في لغة البرمجة العربية";
}
```

هكذا يمكننا استخدام اللغة العربية الفصحى لكتابة كود برمجي، وإنشاء لغة برمجة بالعربية مفهومة وقابلة للقراءة بواسطة مجموعة أوسع من الأشخاص.
