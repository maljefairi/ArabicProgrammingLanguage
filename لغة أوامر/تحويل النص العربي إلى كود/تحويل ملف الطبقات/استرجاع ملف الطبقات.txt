# تحويل ملف الطبقات إلى كود جملة صحيحة باللغة العربية الفصحى:

`قم بتحويل الملف المطلوب إلى الشيفرة المقابلة`

وهذا مثال على استخدامها:

```python
def تحويل_الملف_إلى_الشيفرة(ملف_الطبقات):
    with open(ملف_الطبقات, 'r', encoding='utf-8') as ملف:
        الشيفرة = ملف.read()
    return الشيفرة

الشيفرة_المطلوبة = تحويل_الملف_إلى_الشيفرة("طبقات.txt")
print(الشيفرة_المطلوبة)
```
 في هذا المثال، تم استعمال الدالة `تحويل_الملف_إلى_الشيفرة` لفتح ملف الطبقات الموجود في الحاسوب وقراءته، وبعدها تم عرض الشيفرة المقابلة للملف على الشاشة.

# استرجاع ملف الطبقات:

`قم بالبحث في النظام والوصول إلى ملف الطبقات المطلوب واستعادته`

وهذا مثال على استخدامها:

```python
import os

def استرجاع_الملف(مسار_الملف):
    if os.path.exists(مسار_الملف):
        with open(مسار_الملف, 'r') as ملف:
            البيانات = ملف.read()
        return البيانات
    else:
        return "الملف غير موجود"

البيانات = استرجاع_الملف("/المسار_الى_الملف/ملف_الطبقات")
print(البيانات)
```
في هذا المثال، يتم استخدام الدالة `استرجاع_الملف` لفحص ما إذا كان الملف موجودًا ثم فتحه وقراءة البيانات الموجودة فيه وطباعتها. إذا لم يكن الملف موجودًا ، يتم طباعة رسالة تفيد بأن الملف غير موجود.
