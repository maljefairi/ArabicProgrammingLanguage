أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# إنشاء لغة برمجة باللغة العربية الفصحى 
لغة برمجة باللغة العربية الفصحى هي لغة برمجة تتيح للمطورين كتابة وفهم الأوامر البرمجية باللغة العربية. الهدف من إنشاء هذه اللغة هو تمكين المطورين الذين يتحدثون العربية من البرمجة بكفاءة وفعالية.

## تحويل النص العربي إلى كود
لتحويل النص العربي إلى كود، نحتاج إلى تحديد التعليمات البرمجية بشكل صحيح. على سبيل المثال، تعليمة "بالضبط" يمكن أن توافق الأمر "إذا" في لغات البرمجة الأخرى.

1. **تحويل الأوامر البسيطة**

لنفترض أننا نريد تحويل الأمر "طباعة" إلى كود، يمكن أن يكون شيء مثل:
```arabic
طباعة("مرحبا بك في لغة البرمجة العربية");
```
وهو يعادل التالي في اللغة الإنجليزية: 

```javascript
print("Welcome to Arabic Programming Language");
```

2. **تحويل الجمل الشرطية**

نستخدم الأمر "إذا"، "ثم" و "غير ذلك", وهو يعادل في البرمجة الإنجليزية الأمر "if"، "then"، "else".

```arabic
إذا (العمر >18) {
    طباعة ('أنت بالغ');
} غير ذلك {
    طباعة ('أنت قاصر');
}
```
ورمز الجافاسكريبت المعادل سيكون:

```javascript
if (age > 18) {
    print('You are an adult');
} else {
    print('You are a minor');
}
```
## تصحيح الأخطاء
أدوات تتبع الأخطاء مهمة لأي لغة برمجة. هذه الأداة تساعد على تتبع مصادر الأخطاء وإصلاحها. لدينا بالفعل الأداة "تصحيح الأخطاء" في لغة البرمجة العربية.

```arabic
تحاول {
    // كود برمجي قد يؤدي إلى حدوث خطأ
} التقاط (الخطأ) {
    طباعة("خطأ مكتشف: " + الخطأ);
}
```
يكون البرمجة باللغة الإنجليزية:

```javascript
try {
    // Some code that can lead to a error
} catch (e) {
    print("Error detected: " + e);
}
```
في هذه اللغة، يتم استخدام الكلمات العربية لوصف الأوامر البرمجية، مما يساهم في تحسين وضوح الكود للمستخدمين الذين يتحدثون اللغة العربية كلغة أم.
