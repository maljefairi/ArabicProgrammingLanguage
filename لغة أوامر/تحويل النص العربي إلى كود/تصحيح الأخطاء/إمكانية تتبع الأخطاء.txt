# إنشاء لغة برمجة باللغة العربية الفصحى

تتميز اللغة العربية بتعقيدها ورقتها، مما يجعلها خيارًا أقل شعبية للغات البرمجة. ولكن، الغة العربية الفصحى حديثًا تم استخدامها في تطوير طرق جديدة لبرمجة الحاسوب.

## الكود البرمجي المقترح

تم تصميم الكود ليكون قريب ما يمكن للشكل الطبيعي للجمل العربية.

```arabic
اذا (العمر > 18) {
    طباعة("أنت بالغ")
} والا {
    طباعة("أنت طفل")
}
```

## الشرح

الجملة البرمجية تعبر عن عملية شرطية في البرمجة. "اذا" تتبعها شرط تحققه، في هذا السياق، إن كان العمر أكبر من 18. إذا تحقق الشرط، يتم تنفيذ الكود داخل الأقواس `{}`، ويتم طباعة "أنت بالغ". في حالة عدم تحقق الشرط، يتم تنفيذ الكود بعد كلمة "والا"، ويتم طباعة "أنت طفل".

## مثال على التطبيق

نفترض أن لدينا برنامج يطلب من المستخدم إدخال عمره، وبناءً على العمر الذي يدخله البرنامج سيقوم بإخبار المستخدم إذا كان بالغًا أم لا. البرنامج قد يكون كالتالي:

```arabic
عمر المستخدم = أدخل("كم عمرك?")
اذا (عمر المستخدم > 18) {
    طباعة("أنت بالغ")
} والا {
    طباعة("أنت طفل")
}
```

بهذا الشكل، البرمجة باللغة العربية تبدأ بأخذ مكانتها في صناعة البرمجة.
