أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

بالتأكيد, هنا مثال على الأمر البرمجي باللغة العربية الفصحى: 

```عربية قصيدة
أنشئ وظيفة بأسم "فحص_الروابط" {
   تحتاج الى "قائمة_الروابط";
   أنشئ متغير بأسم "روابط_معطوبة";

   لكل "رابط" في "قائمة_الروابط" {
      إذا كان "رابط" غير_صالح {
         أضف "رابط" الى "روابط_معطوبة";
      }
   }
   
   أعد "روابط_معطوبة";
}
```
هذا الكود بالباسكال يقوم بإنشاء وظيفة باسم "فحص_الروابط" التي تستقبل قائمة الروابط كمعلمة ومن ثم يتم التحقق منها. إذا كان الرابط غير صالح تكون روابط الخطأ .
