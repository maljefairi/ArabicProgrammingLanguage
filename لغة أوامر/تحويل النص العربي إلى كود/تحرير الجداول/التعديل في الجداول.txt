أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# الادراج في الجداول
الأمر البرمجي لإدراج بيانات في الجداول هو "أدخل". يتم استخدامه بالطريقة التالية:

```
أدخل في الجدول اسم_الجدول { الحقل1: القيمة1، الحقل2: القيمة2، الحقل3: القيمة3 }
```

مثال على الاستخدام:

```
أدخل في الجدول الموظفين { الاسم: "علي", الراتب: 2000, الوظيفة: "مهندس" }
```

# التعديل في الجداول
الأمر البرمجي لتعديل بيانات في الجداول هو "عدل". يتم استخدامه بالطريقة التالية:

```
عدل في الجدول اسم_الجدول { الحقل1: القيمة1، الحقل2: القيمة2 } حيث { الحقل3: القيمة3 }
```

مثال على الاستخدام:

```
عدل في الجدول الموظفين { الراتب: 3000 } حيث { الاسم: "علي" }
```

# الحذف من الجداول
الأمر البرمجي لحذف بيانات من الجداول هو "احذف". يتم استخدامه بالطريقة التالية:

```
احذف من الجدول اسم_الجدول حيث { الحقل1: القيمة1 }
```

مثال على الاستخدام:

```
احذف من الجدول الموظفين حيث { الاسم: "علي" }
```

# تغيير هيكل الجداول
الأمر البرمجي لتغيير هيكل الجداول هو "غير_هيكل". يتم استخدامه بالطريقة التالية:

```
غير_هيكل الجدول اسم_الجدول { الحقل1: نوع_البيانات1، الحقل2: نوع_البيانات2 }
```

مثال على الاستخدام:

```
غير_هيكل الجدول الموظفين { الراتب: عدد_صحيح }
```
