أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

إذا أردت إنشاء لغة برمجة تستخدم الجمل الصحيحة، يمكنك استخدام التالي:

**تعريف الكائنات:**

تستطيع تعريف الكائنات في اللغة العربية بعبارة "أعرف كائن" متبوعة بإسم الكائن.

مثلا:

```
أعرف كائن سيارة
```
تهيئة الكائن يتم بإستخدام العبارة "أنشئ كائن" وإسم الكائن:

```
أنشئ كائن سيارة
```

**التعامل مع الكائنات:**

إذا أردنا ضبط خاصية للكائن نستخدم العبارة "ضبط" متبوعة بإسم الكائن واسم الخاصية والقيمة المراد ضبطها.

مثلا:

```
ضبط سيارة.لون = أحمر
```

**الإدخال والإخراج:**

يمكنك اختيار الإدخال باستخدام الأمر "أدخل" متبوعا بالمتغير الذي تريد أن يأخذ القيمة، وكذلك استخدام الأمر "أطبع" للإخراج.

مثلا:

```
أدخل اسم
أطبع "مرحبا " + اسم
```

لاحظ أن هذه لغة برمجة وهمية ولا تعمل في الواقع. الهدف منها هو فقط ليصور كيف يمكن للغة برمجة باللغة العربية أن تبدو.
