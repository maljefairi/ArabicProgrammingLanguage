أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

**تحويل البرمجة الكائنية**

بفضل لغة البرمجة باللغة العربية، يمكننا تحويل البرمجة الكائنية ببساطة وفعالية. بدءا من تعريف الكائنات إلى التعامل معها وحتى الخواص المتقدمة مثل الوراثة والتعدد الشكلي. أليك كيف يبدو:

- تعريف الكائنات :

   ```العربية
   نوع كتاب{
      عنوان: العربية,
      مؤلف: العربية,
      سنة النشر: الرقم,
      }
   ```

- التعامل مع الكائنات :

  ```العربية
      كتابي = كتاب جديد{عنوان: "الأيام", مؤلف: "طه حسين", سنة النشر: 1935}
  ```
- الوراثة :

  ```العربية
      نوع كتاب أكاديمي : كتاب {
      المادة : العربية
      }
  ```      

**القوائم والمصفوفات**

لتحويل القوائم والمصفوفات، المفتاح هو التركيز على التفاصيل الدقيقة. الأمر بسيط للغاية:

- إنشاء قائمة:

   ``` العربية
   قائمة = [1، 2، 3، 4، 5]
   ```

- الوصول إلى عنصر في القائمة:

   ``` العربية
   العنصر_الأول = قائمة[0]
   ```

- تحديث عنصر في القائمة:

   ``` العربية
   قائمة[0] = 100
   ```

- إنشاء مصفوفة:

   ``` العربية
   المصفوفة = [[1، 2، 3]، [4، 5، 6]، [7، 8، 9]]
   ```
  
- الوصول إلى عنصر في المصفوفة:
   
   ``` العربية
   العنصر_الأول = المصفوفة[0][0]
   ``` 

هذا مبدئياً، إن المفهوم للبرمجة الكائنية والقوائم والمصفوفات يطبق جميعًا باللغة العربية الفصحى، مما يجعل العملية تتم بسلاسة وكفاءة.
