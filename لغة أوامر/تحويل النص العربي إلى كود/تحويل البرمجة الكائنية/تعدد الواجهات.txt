أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

## تعريف الكائنات

في البرمجة الكائنية، الكائن هو وحدة برمجية تحتوي على حالة وسلوك. الحالة يتم تعريفها بواسطة المتغيرات والسلوك يتم تعريفها بواسطة الوظائف. في لغتنا البرمجية، يمكننا تعريف الكائن عن طريق استخدام الكلمة الأساسية "كائن".

```
كائن السيارة {
    متغير اللون = "أحمر";
    متغير الموديل = 2022;

    وظيفة قيادة() {
        طباعة("السيارة تقود الآن");
    }
}
```
وهنا قمنا بتعريف كائن جديد يسمى "السيارة" ولديه حالتين هما "اللون" و"الموديل" بالإضافة إلى سلوك واحد هو "قيادة". يمكننا الآن استخدام هذا الكائن في برنامجنا، على سبيل المثال، يمكننا إنشاء سيارة جديدة وقيادتها كما يلي:

```
كائن سيارتي = جديد السيارة();
سيارتي.قيادة();
```

## تعدد الواجهات
في البرمجة الموجهة للكائنات، الواجهة هي نوع من الكائنات الذي يعرف مجموعة من السلوكيات التي يجب على الكائن تنفيذها. في لغتنا البرمجية، يمكننا تعريف الواجهة باستخدام الكلمة الأساسية "واجهة".

```
واجهة القيادة {
    وظيفة ابدأالقيادة();
    وظيفة اوقفالقيادة();
}

كائن السيارة ينفذ القيادة {
    وظيفة ابدأالقيادة() {
        طباعة("بدأت القيادة");
    }

    وظيفة اوقفالقيادة() {
        طباعة("تم الإيقاف");
    }
}
```
وأصبح الأن لدينا كائن "السيارة" ينفذ واجهة "القيادة" وبالتالي يجب على "السيارة" تنفيذ كل الوظائف المعرفة في "القيادة".
