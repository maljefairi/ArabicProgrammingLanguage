# تحويل البرمجة الكائنية في اللغة العربية

**1. تعريف الكائنات:**
في سياق البرمجة الكائنية، الكائن هو وحدة تحتوي على بيانات وأساليب للتعامل مع هذه البيانات.

```عربي
كائن سيارة
```

**2. التعامل مع الكائنات:**
يمكننا التعامل مع الكائنات وتنفيذ عمليات مثل تعيين القيم واستدعاء الأساليب.

```عربي
سيارة.ركن()
```

**3. الوراثة:**
تسمح العملية البرمجية الوراثة للكائنات بتجنب التكرار وزيادة الإعادة الاستخدامية والكفاءة.

```عربي
كائن سيارة_رياضية تورث سيارة
```

**4. التعدد الشكلي:**
تركز البرمجة الكائنية على التعدد الشكلي، الذي يتيح استخدام واجهة موحدة لكائنات من أصناف متعددة.

```عربي
سيارة_رياضية.ركن()
```

**5. التجميع:**
تجمع بعض الكائنات الأخرى كجزء من كائن. 

```عربي
كائن سائق {
   سيارته: سيارة
   . . . 
}
```

**6. التنفيذ الشرطي:**
التنفيذ الشرطي هو قرار يتم اتخاذه على أساس حقيقة ما، وسوف يتم تنفيذ بعض الأوامر عندما يتم تنفيذ الشرط.

```عربي
إذا سائق.عمر < 18: سيارة.قف()
```

**مثال على التطبيق:**
اذا أردنا تطبيق برنامج بسيط بلغتنا البرمجية العربية التي تمثل سيارة تحتوي على بعض الخصائص مثل اللون والموديل وتحتوي على بعض الأساليب مثل القيادة والتوقف، ونريد فحص إذا كان السائق أقل من 18 عاماً، في هذه الحالة سنمنعه من قيادة السيارة.
يمكننا القول:

```عربي
كائن سيارة {
   لون: أسود
   موديل: 2020
   قد_للأمام(): ...
   توقف(): ...
}

كائن سائق {
   اسم: "أحمد"
   عمر: 17
   سيارته: سيارة
}

إذا سائق.عمر < 18: سيارة.توقف()

```
