# البرمجة باللغة العربية الفصحى

في اللغة البرمجية العربية، يمكننا كتابة الأوامر بطريقة مفهومة وبسيطة. الأمر الأساسيّ في البرمجة الكائنية هو تعريف الأكائن. سأضع أمثلة توضح عمليات البرمجة الكائنية باللغة العربية الفصحى.

1. **تعريف الكائنات:** 

يمكننا تعريف الكائنات بالطريقة التالية:

```العربية
أكائن سيارة:
    عناصر:
        لون
        نوع
```
هذا الأمر بسيط وواضح، نحن نعرف كائن باسم "سيارة" لديها عناصر هي "اللون" و "النوع".

2. **التعامل مع الكائنات**:

يمكننا التعامل مع الكائن الذي أنشأناه بالطرق التالية:

```العربية
سيارتي = سيارة جديدة
سيارتي.لون = أحمر
سيارتي.نوع = سدان
```
في الأمثلة أعلاه، نحن نعرّف سيارة جديدة باسم "سيارتي"، ثم نعيّن اللون لها بـ"أحمر" والنوع بـ"سدان".

3. **التعدد الشكلي**:

في البرمجة، التعدد الشكلي يسمح لنا بتوفير واجهة واحدة لأنواع مختلفة من الأداء. سنقوم بتعريف دالة "بدأ" يمكنها بدء السيارة أو الدراجة بناءًا على الكائن.

```العربية
أداء بدأ(نقل) {
    إذا نقل == سيارة:
        اطبع( تشغيل السيارة )
    وإلا إذا نقل == دراجة:
        اطبع(تشغيل الدراجة)
}
بدأ(سيارتي)
```

بتطبيق هذا الأمر البرمجي، سيتم طباعة "تشغيل السيارة" عند التعامل مع السيارة و"تشغيل الدراجة" عند التعامل مع الدراجة.
