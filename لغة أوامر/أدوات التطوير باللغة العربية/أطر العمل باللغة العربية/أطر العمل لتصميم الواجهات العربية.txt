أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# أطر العمل لتصميم الواجهات العربية
أطر العمل لتصميم الواجهات العربية هي أدوات تساعد المطورين على إنشاء تطبيقات تفاعلية باللغة العربية. تسمح هذه الأدوات بتصميم الواجهات بطريقة مبسطة ومباشرة. فيما يلي مثال على كود برمجي:

```arabic
أطار("ورقة قائمة") {
    مركز() {
       عنوان(نص="مرحبا بك في تطبيقنا") 
    }
    زر(نص="أنقر هنا") {
        دالة(أمر) {
            إطبع(نص="تم النقر على الزر.") 
        }
    }
}
```

في الكود السابق، نحن نستخدم الأمر "أطار" لدعم التطبيق و "مركز" لتوسيط العناصر داخل الأطار. خلقنا فيه خط كبير حيث نستخدم الأمر "عنوان" ونضيف النص "مرحبا بك في تطبيقنا". ثم نستخدم الأمر "زر" لإنشاء زر وضيافة نص "أنقر هنا" عليه. عندما ننقر فوق الزر، يتم طباعة رسالة "تم النقر على الزر." الأمر "دالة" يستخدم لتحديد السلوك بعد النقر على الزر.
