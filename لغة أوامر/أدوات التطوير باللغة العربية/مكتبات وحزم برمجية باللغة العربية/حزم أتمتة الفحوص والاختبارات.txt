أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

```markdown
نظراً لكون اللغة البرمجية العربية غير موجودة حاليًا، سنقدم الأمثلة الوهمية التالية لتوضيح أفكار حول كيفية ظهور كود باللغة العربية الفصحى.

# المثال الأول: تعريف المتغيرات
في اللغة البرمجية العربية، قد يكون تعريف المتغيرات بهذا الشكل:

```
متغير السن = 25
```
تعطي هذه الجملة المساحة لمتغير يدعى "السن" وتعين قيمتها إلى 25

# المثال الثاني: الدوال
نعرف دالة جديدة يكون الكود كالتالي:

```
دالة جمع(أ, ب) {
   عودة أ + ب
}
```
هذه الدالة تأخذ في الاعتبار متغيرين، تُجمعهما، وتُعيد القيمة

# المثال الثالث: الشروط
في تنفيذ الأوامر بناءً على الشرط، قد يكون الكود كالتالي:

```
إذا (السن > 18) {
   اكتب("الشخص بالغ")
} وإلا {
   اكتب("الشخص قاصر")
}
```
هذا الكود يقوم بالتحقق من الشرط (إذا كانت السن أكبر من 18)، في هذه الحالة يقوم بإرجاع "الشخص بالغ"، وإلا يرجع "الشخص قاصر".

يرجى ملاحظة أن هذه فقط أمثلة وهمية لما يمكن أن تبدو عليه اللغة البرمجية العربية، وهي غير قابلة للتنفيذ في الواقع.
```
