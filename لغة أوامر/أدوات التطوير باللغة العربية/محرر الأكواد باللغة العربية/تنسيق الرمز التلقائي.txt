أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# تنسيق الرمز التلقائي
تنسيق الرمز التلقائي هو أحد الميزات المهمة في البرمجة، حيث يهدف إلى تحسين قراءة وفهم الكود البرمجي، من خلال اتباع مجموعة من الأنماط والقواعد البرمجية.

في لغتنا البرمجية العربية نستخدم الأمر "صفّ بوقتك" لإجراء عملية تنسيق الكود البرمجي.
## مثال على استخدام الأمر "صفّ بوقتك":

بفرض أن لدينا البرنامج البسيط التالي:

```
إنشاء دالة  جمع(أ, ب)
{
دع النتيجة = أ  + ب
أرجع النتيجة
}
```
إذا تم تنفيذ الأمر  "صفّ بوقتك"، سيتم تنسيق الكود البرمجي بشكل تلقائي ليصبح بشكل الأتي:

```
إنشاء دالة  جمع(أ, ب)
{
    دع النتيجة = أ  + ب
    أرجع النتيجة
}
```
حيث تم تحقيق أفضل تنظيم للكود من خلال تحويل التباعد والتعداد الهرمي بشكل تلقائي.
