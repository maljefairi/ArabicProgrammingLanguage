أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

كود البرمجة باللغة العربية يمكن أن يتم بالطريقة التالية:

1. إنشاء متغير (variable): 

   تضع كلمة "متغير" أمام الاسم المراد استخدامه للمتغير وبعد ذلك تضع القيمة التي تريد تخزينها في المتغير.

   مثال: 
   ```
   متغير اسم = "عمر";
   ```

2. الدوال (functions): 

   تبدأ بكلمة "دالة" ثم اسم الدالة المراد إنشائها وبعدها تكتب عبارات الدالة.

   مثال:
   ```
   دالة طباعة(رسالة){
     طباعة(رسالة);
   }
   ```

3. الشرطيات (if statements):

   تبدأ بكلمة "إذا" ثم الشرط وبعد ذلك تكتب العبارات التي تود تنفيذها إذا تحقق الشرط.

   مثال:
   ```
   إذا (عمر > 20){
     طباعة("العمر أكبر من 20");
   }
   ```

4. الحلقات (loops):

  تبدأ بكلمة "بينما" ثم الشرط وبعدها تكتب العبارات التي تود تنفيذها طالما الشرط صحيح.

   مثال:
   ```
   متغير i = 0;
   بينما (i<10){
     طباعة(i);
     i = i + 1;
   }
   ```

تبقى الواجهة المستخدم باللغة العربية حيث يتم استبدال التعليمات البرمجية بكلمات عربية تحقق نفس الوظيفة. لذا، يجب عليك فقط تعلم الكلمات البرمجية العربية المناسبة.
