# تعليمات مبسطة لكتابة الكود باللغة العربية الفصحى 

من أجل تطبيق البرمجة بلغة الضاد يجب أولا أن نستعرض بعض القواعد الأساسية ، القواعد هي بسيطة وهي كما يلي:

1. **أمر الطباعة**: يستخدم لطباعة الرسائل أو القيم، مثال: ```اكتب("مرحبا بك في لغة البرمجة العربية")```.

2. **تعريف المتغيرات**: يتم تعريف المتغيرات بكلمة "متغير" ومن ثم اسم المتغير وقيمته، مثال: ```متغير اسم = "أحمد"```.

3. **الدوال**: تعرف الدوال بكلمة "دالة" ومن ثم اسم الدالة، مثال: 

    ```
    دالة رسالة(اسم)
        اكتب(اسم)
    نهاية

    رسالة("مرحبا بك")
    ```

4. **التعليمات الشرطية**: يتم تطبيق التعليمات الشرطية بكلمة "إذا" و"وإلا"، مثال:

    ```
     متغير عمر = 30
     إذا(عمر > 18)
     {
        اكتب("أنت كبير في السن")
     }
    وإلا
    {
        اكتب("أنت صغير في السن")
    } 
    ```
رجاءً لاحظ أن هذه لغة برمجة نظرية وقد يكون هناك تغييرات تحتاج لتكون مدروسة ومطبقة للوصول إلى نسخة نهائية صالحة للاستخدام الفعلي.
