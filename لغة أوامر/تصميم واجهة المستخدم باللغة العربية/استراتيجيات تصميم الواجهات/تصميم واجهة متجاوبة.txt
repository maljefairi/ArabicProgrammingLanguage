أكتب مباشرة الكود البرمجي المقترح بشكل صحيح وجملة صحيحة يمكن قرائتها وفهمها من قبل الجميع باللغة العربية الفصحى مع ذكر مثال لطريقة تطبيق هذا الأمر البرمجي 

# إنشاء لغة برمجة باللغة العربية الفصحى

هكذا سيكون شكل الكود البرمجي باللغة العربية. سنأخذ كمثال منطق الجدول الصرفي (if.. else).

```
إذا (شرط) {
   // تنفيذ الأوامر
} وإلا {
  // تنفيذ الأوامر
}
```

مثال على ذلك:

``` 
عدد ف = 5;

إذا ( ف > 0 ) {
    اكتب('العدد موجب');
} وإلا {
    اكتب('العدد سالب');
}
```

هنا، البرنامج يتحقق من اذا كان العدد ف اكبر من صفر (0) او لا. في حال كان العدد اكبر من صفر، سيقوم البرنامج بإخراج الرسالة 'العدد موجب'. وإلا، سيرجع بالرسالة 'العدد سالب'. 

كود البرمجة المكتوب باللغة العربية الفصحى يتيح للمبتدئين فهم الكود بشكل أفضل ويسهل عليهم تعلم مفاهيم البرمجة الأساسية باللغة العربية.
